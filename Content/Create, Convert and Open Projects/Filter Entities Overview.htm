Filter Entities Overview

You can filter to entities using several methods.

  • To filter to entities you have selected in the Entity Explorer, see Filter to a Selection of Entities.
  • To filter to wells you have flagged, see Flag and Filter Entities.
  • To create a filter query, see Create a Filter Query.
  • To filter to a UWID list you have in a text file or Excel file, see Filter to a UWI List.

Notes on Applying Filters

To clear filters, see Clear Filters.

When a filter is applied to the project, the Filter Well List icon is red .

If you create entities or edit entity properties that affect the filter results, the results are not automatically updated. Use the Reapply Filter button () to update the results.
