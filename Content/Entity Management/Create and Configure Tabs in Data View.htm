Create and Configure Tabs in Data View

When you display the Data View, default tabs are displayed. In addition to the default tabs, you can create your own Group Tab (a set of tabs consisting of a top level tab and one or more subtabs).

The procedure below explains how to create a Group Tab and configure how data is displayed.

To create a Group Tab (and additional subtabs) in the Data View

  1. Click  next to the Well Views tab.
  2. In the Tab Group Name dialog box, type a name for the tab group and click OK.
    A top level tab with the name you entered is created. Also, a subtab with the default name Data View is created.
  3. Under Select an input area for the tab, select the type of data input you want to display on the tab.
    The name of the subtab changes to the type of data you selected.
  4. In the layout panel on the right, do any of the following, as required:
    5. To Do this
    Display the layout panel, On the right of the screen, click  .
    Rename a subtab, In the Tab Name field, type a new name.
    Hide values on the grid, Under Values, deselect the values you want to hide.
    Move column values to rows, Drag fields to the Rows window.
    Move row values to columns, Drag fields to the Columns window.
    Change the filter (such as Plan or Reserves Category), Under Filters, click the arrow next to the filter icon () and select a new value.
    Apply a layout template, See Apply a Layout Template to a Tab in Data View.
    Include child entities Displays child entities when selected.
    Defer grid updating, See Defer Grid Updating in Data View.
    Save the layout as a template, See Save a Tab Layout as a Template in Data View.
    When you click outside the layout panel, the layout changes are automatically applied.
  5. Add more subtabs, if required, by clicking  under the first subtab and repeating the steps above.
