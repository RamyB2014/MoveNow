Set the Project Currency

The project currency is used for economic calculations performed at the folder and project level. You can specify a different currency for entities. has three currencies by default: US dollar, Canadian dollar, and the Euro.

The currencies available in the Project Currency list are limited to the currencies already created in the project. If the currency you want to use is not listed in the Project Currency list, see Add a Currency to create it.

To set the project currency

  1. On the Tools menu, click Options.
  2. In the Options dialog box, click the Project Options tab.
  3. On the left, select General.
  4. On the right, select the project currency from the Project currency list and click OK.
