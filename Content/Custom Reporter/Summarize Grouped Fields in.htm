Summarize Grouped Fields in Custom Table Reports

Grouped fields can be displayed as Expanded or Summarized. Summarizing a group allows you to view a summary of the items to report.

For example, if the group Date is summarized, the report data will be organized by year. If it is expanded, it will be displayed by month.

To summarize a group

  1. In the Groups window, right-click a group field, point to Expand Group and select Summarized. The summarized group field displays with a blue sigma sign.
  2. In the Summary Method row of your report, select a summary method for the data that is being summarized.
If you do not select a summary method from the Summary Method row in the report outline at the bottom of the Edit Current Object tab, data will not be displayed.

Group Fields can also be nested within each other. This allows you to group your results by two or more fields. See Group Fields in Custom Table Reports.
