Cross Plot Report Overview

On Cross Plot | Distribution, you can create a Cross Plot report from a selected set of entities, such as a Type Well or a Roll up. 

There are three tabs beneath the Cross Plot tab:

  • Distribution: A table with the statistical data as related to an initial production rate (IP) and an estimated ultimate recoverable volume (EUR) is provided. A Cumulative Distribution graph is provided plotting IP and EUR against the cumulative probability on a semi-log plot.
  • Frequency: The frequency distribution plots of IP and EUR demonstrate whether the distribution is lognormal or otherwise.
  • Probability: The probability of the IP and EUR are plotted on separate graphs. Displayed trends can be used to determine the applicability of the selected set of wells as a like set of wells.
