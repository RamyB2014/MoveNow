Assign Roles to Users

A role is a group of policies (permissions). By creating roles and then assigning the roles to users, you can quickly apply a consistent set of permissions to several users. You can also use roles in combination with individual policies to assign a unique set of permissions to one user. See Assign Policies to Users.

To assign a role to a user

  1. From the Administration menu, select Security Settings.
  2. On the Users tab, select a user.
  3. Click Assign Role.
  4. In the Select Roles window, select a role or roles to assign to the user. Hold the Ctrl or Shift key to select multiple roles.
  5. If assign the Edit Entity or View Entity policies to the role, create a filter to specify which wells the user can view or edit.
  6. Click OK.
