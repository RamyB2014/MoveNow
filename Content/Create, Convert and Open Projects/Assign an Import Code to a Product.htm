Assign an Import Code to a Product

Common products such as condensate, oil, and gas are assigned import codes in by default. However, if you have added a custom product and you are able to import production or injection data from a data vendor such as IHS or geoScout, you need to assign it an import code.

To assign an import code to a product

  1. In the Tools menu, point to Global Project Data and click Products.
  2. Click the Import Codes tab.
  3. Next you the custom product, enter the code in the Import Codes column.
  4. Click OK.
