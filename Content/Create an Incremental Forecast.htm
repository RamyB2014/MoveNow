Create an Incremental Forecast

On Predictions | Declines, the Forecast Mode enables you to create an incremental forecast using your existing “base” forecast as a starting point. In the Incremental mode, you only need to define the incremental decline segment rather than recreating the entire forecast.

In Incremental mode, you can set a transition date and then increase production at that date with a relative or an absolute value.

You can also assign the forecast incremental costs (see Enter Incremental Costs) and a project start date. You can then move the project start date on the Timeline tab to shift the start of the incremental forecast and associated costs together.

To create an incremental forecast

  1. On Predictions | Declines, in the PDP reserves category, select the Incremental forecast mode (above the decline calculator).
  2. Select the PD reserves category and add a forecast.
  3. Click the header above the forecast parameter to open the decline calculator.
    The Base Forecast & Incremental Parameters are displayed on the left. The incremental forecast is displayed on the right.
  4. In the decline parameters on the right, adjust the incremental forecast’s start date.
    The start date becomes the Transition time in the Base Forecast. The Base Forecast’s parameters are displayed below, as of the transition date.
  5. Do one of the following:
    6. To Do this
    Increase the incremental forecast by a relative value, In the Base Forecast & Incremental Parameters, enter a percentage in the ∆Qi.
    Increase the incremental forecast by an absolute value, In the decline parameters on the right, edit the Qi other parameters.
