Map Coordinates

The Map requires all latitude/longitude coordinates to be in NAD83 coordinates and represent the bottom hole location of the well. 

If you export well information from another application for use in , export the coordinates in NAD83 coordinates if there is an option to do so.

You can also update well data from an online source such as IHS DataHub or geoLOGIC Data Centre, which will update the coordinates to NAD83 if you select the Import latitude/longitude in ’s User Options (Tools > Options > User Options > Import Parameters—General).
