Edit Meter Stations

In the Meter Station Editor, you can edit, add, or remove meter stations.

To edit, add, or remove meter stations
  1. On the Tools menu, point to Global Project Data, and select Meter Stations.
  2. Do one of the following:
    3. To Do this
    Edit a meter station, Select a meter station from the list and edit the Meter Station Properties.
    Add a meter station, Click Add. Enter a name and click OK. Edit the Meter Station Properties.
    Remove a meter station, Select a meter station from the list and click Remove.
