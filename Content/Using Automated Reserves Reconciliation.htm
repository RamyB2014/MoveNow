Using Automated Reserves Reconciliation

To use automated reserves reconciliation (ARR), complete the following steps first:

  • Import opening balances using the Spreadsheet Import. This step is required if you are not working with a posted project. The opening balances are required so can compare the current value of entities. You can use ARR without opening balances, but you must assign each entity a Change Reason.
  • Enable Automated Reserves Reconciliation
  • Create or Edit Change Record Categories or Summary Types
  • Specify a Change Record Sequence
  • Specify a Change Reason on Wells
  • If you have completed the steps above, see Auto Reconcile Wells.
