Use Proxy Names for Spreadsheet Import

In many cases you will import data by UWI. However, you can also target entities by specifying a proxy name instead of a UWI. Proxy names include Vendor ID, API Number, Field, Field Code, Pool, Pool Code, Spreadsheet Fields, or Import Tag. Using a proxy name instead of a UWI can save you time by eliminating the need to create a well list. For example, if you want to add the same capital cost to all wells in Pool 1, you can simply create a row in your spreadsheet for Pool 1 and then specify Proxy name- Pool in the Spreadsheet Import tool. All wells in that pool will be updated with the new capital cost.

If you want to use Vendor IDs to import data, you must first copy the ID into the Spreadsheet Import column of the Vendor IDs dialog box. See Use Vendor IDs in the Spreadsheet Import.
