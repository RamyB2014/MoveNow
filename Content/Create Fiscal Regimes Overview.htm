Create Fiscal Regimes Overview

includes fiscal regime data for Canada and the United States. This regime data is used when you set up interests for a Canadian or American entity in Economics | Interest & Royalties. However, if you are drilling outside of Canada or the United States or you want to customize the existing regimes, you can set up or edit custom regimes for any country, province, or state.

Creating a regime involves these basic steps:

  1. Create a new regime or copy an existing regime. Add regime information such as the type (concession or contract, before tax or tax) county, currency, etc.
  2. Add entity-level and regime-level inputs.
  3. Create formulas for the regime variables.
  4. Preview the regime.
