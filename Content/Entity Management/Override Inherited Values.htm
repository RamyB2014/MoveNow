Override Inherited Values

Tabs with values inherited from Common display a yellow message bar.

Tabs with values inherited from Common through intervening plans/reserves categories or from other plans and reserves categories display a grey message bar with breadcrumbs displaying the inheritance chain.

To override the data inheritance and enter values for the current plan and reserves category, click Override.

Use Edit Source to quickly go to the data source (Working-Common, in this example), edit it, and return to the current plan/reserves category. The current tab will inherit the values you entered on the source tab.

See Understanding the Common Reserves Category and Reset Overridden Values.
