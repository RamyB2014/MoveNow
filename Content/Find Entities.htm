Find Entities

The Find function can be used to locate wells in a project or to find wells when you have moved from one Explorer view to another (e.g. from Entity Hierarchy to Wells).

The Find function can be accessed by:

  • Selecting Find from the Edit menu
  • Right-clicking in the Explorer and selecting Find
  • Pressing F3 (for new searches only). See Using the F3 Key to Find Wells, below.)

To find a well

  1. On the Edit menu, click Find.
  2. Select the search format you want to use.
  3. Enter the well name or UWID (an asterisk can be used as a wildcard).
  4. Click Find Next or Find All.

When the well is found, it is highlighted in the Explorer and it is listed in the bottom of the Find dialog box.

Using the F3 Key to Find Wells

Once you complete a search in the Find dialog box, it is saved in memory until another search is entered or is shut down. Pressing the F3 key performs the saved search without reopening the Find dialog box. If the saved search returned more than one well, pressing F3 scrolls through the wells in the same way as Find Next.

Pressing Ctrl+F3 saves the current well to the Find memory so that the same well can be found quickly again later. For example, if you are in the Entity Hierarchy view and press Ctrl+F3, then later you are in the Data View and want to find the same well, pressing F3 will find the well.

If the Find dialog box is open when you press the F3 key to save a well, the well will not be saved to the Find memory. Close the Find box before pressing F3.
