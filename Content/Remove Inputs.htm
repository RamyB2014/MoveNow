Remove Inputs

Use the procedure below to bulk remove inputs (overrides) in the current reserves plan.

Removing inputs reverts the input to the source reserves category and plan.

You can remove inputs on one tab at a time, in one reserves category or plan by clicking Reset in the information bar. See Reset Overridden Values.

To remove plan inputs

  1. Select the reserves category or plan with the inputs you want to remove.
  2. From the Entity menu, select Data Manager and click Remove Inputs.
    3. You can also access the Data Manager from the Summary window: Right-click and select Remove Inputs.
  3. Select the Reserves Categories with inputs you want to remove.
  4. Select the following options, as required:
    5. Option Description
    Apply to children of groups If selected, child wells are also removed.
    Remove all inputs All inputs are removed.
    Remove selected inputs Only selected items are removed.
  5. Click Remove.
