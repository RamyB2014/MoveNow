Validate a Well List

If you want to see if a well or list of wells exists in your project, use the Well Detective. When you enter UWIs or API Numbers into the Well Detective, the Status column indicates if the well is Found or Not Found. You can also filter your project to the found wells, by clicking Filter to Entities.

To validate that wells exist in your project

  1. From the Tools menu, select Well Detective.
  2. In the Well Detective dialog box, select UWI or API Number.
  3. Do one of the following:
    • In the UWI or API Number column, type the number you want to validate and press Enter.
    • Click Import to import a list of numbers (.csv, .txt, or .lst formats). Browse to the file and click Open.
  1. Optional: To filter to the found wells, click Filter to entities.
