Set a Variance Baseline for Remaining Gas or Oil on Predictions | Workspace

When you are viewing data on Predictions | Workspace, you can set the Remaining Gas or Oil for one forecast as the baseline, and view the Remaining Gas or Oil variance from that baseline for all other forecasts.

To set a forecast as the baseline, right-click the row and click Set Row as Variance Baseline.

This row will be indicated by a blue arrow in the far left column ().

The Oil/Gas Remaining Variance % column for the baseline forecast will always display 100%. For all other forecasts, the Variance % column will display the variance percentage from the baseline.
