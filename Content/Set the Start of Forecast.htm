Set the Start of Forecast

You can use the Set Start of Forecast dialog box to change the forecast start date on any number of wells. When you change the date, the decline parameters change to represent the new point on the curve.

To move the forecast start date on one well without altering the decline parameters, edit the date on the Summary tab of Predictions | Declines.

To move the forecast start date on several wells without altering the decline parameters, see Adjust Entity Dates.

To set the start of forecast for all the wells in your project, select the highest level folder in the Entity Explorer and follow the procedure below. If you only want to set the start of forecast on some wells, you can select the folder level that contains them or you can filter to them first.

To set the start of forecast

  1. Select the wells you want to edit.
  2. Do one of the following:
    • On the Predictions menu, click Set Start of Forecast.
    • Click  on Predictions | Declines and click Set Start of Forecast
  1. Under Forecast Start Date Setup, choose one of the three forecast start date options.
  2. Under Options, select the Reserves Categories you want to apply the change to.
  3. Confirm the Entity Count to ensure you are best fitting or refitting the correct number of entities.
  4. Click OK.
