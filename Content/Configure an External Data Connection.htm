Configure an External Data Connection

Before you can import or update  monthly production (using PPDM), daily data (using your corporate data hub), or actual capital costs from AFENav, you must configure the IHS, GeoLogic, Daily Data, or AFE vnudl settings in .

To configure an external data connection

  1. On the Tools menu, click Options.
  2. On the User Options tab, click Data Sources.
  3. From the Profile list under Data Source, select GeoLogic, IHS, Daily Data, or Actual Capital Costs (AFENav).
  4. In the Connection field, click .
  5. Browse to the vnudl file you want to use and click Open.
  6. Click Validate to test the connection.
  7. Click OK.

If the validation is not successful, try another vnudl file or contact your administrator.

Ensure the data you want to import is selected under Data to Import in User Options: External Data Settings.
