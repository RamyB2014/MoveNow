Preview Regime Output

This procedure is performed on the Preview tab in the Fiscal Regimes Editor. To open the editor, click the Tools menu, point to Global Project Data and select Fiscal Regimes.

To preview the output of your regime

  1. In the Fiscal Regime Editor, click the Preview tab.
  2. In the Preview Entity list, click  to display the Preview Options dialog box.
  3. In the list at the top of the Preview Options, select the type of entity you want to display and then select the entity.
  4. Select a Plan and Reserves Category.
  5. If you currently have a filter applied to your project and you want to apply it in the preview, click Apply Current Filter.
  6. Click OK.
  7. To view the results for different regime categories, select a Category and Subcategory and click Refresh.
Variables that have not been calculated are displayed in grey text.
