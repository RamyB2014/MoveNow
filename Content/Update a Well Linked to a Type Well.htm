Update a Well Linked to a Type Well

When you have edited a type well’s forecast, you must update any wells linked to the type well because the new type well forecast is not automatically applied to any linked wells. The procedure below explains how to update the type well link for a single well on Predictions | Declines.

To find all the wells linked to a type well and update all the linked wells’ forecasts, see Find and Update Wells Linked to a Type Well.

To update a linked well

  1. Select the well you want to update.
  2. On Predictions | Declines, click  and then click  next to Type Well Link. The Source Type Well is displayed.
  3. Click Apply to apply the updated type well data to the well.
