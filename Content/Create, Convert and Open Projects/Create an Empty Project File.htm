Create an Empty Project File

Project files are single-user, file-based projects that you can store on your computer and email to other users.

To create a project file

  1. On the File menu, click Open.
  2. In the Project Launcher, click Create New Project.
  3. In Project Name, type a name.
  4. In File, browse to a location to save the new project file.
  5. From the Type list, select Project File.
  6. Click OK.

A new project is created in the location you specified and opens. The new project appears in the Project Launcher the next time you open it.
