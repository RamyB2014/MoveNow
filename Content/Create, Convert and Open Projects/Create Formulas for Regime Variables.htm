Create Formulas for Regime Variables

When a formula is used in a variable, the variable’s value is determined by the result of the formula.

You can create formulas for use in your regime variables. The Formula Editor contains Constants, Economic Variables, Functions, Operators, Regime Variables, and Units.

This procedure is performed on the Edit tab in the Fiscal Regimes Editor. To open the editor, click the Tools menu, point to Global Project Data and select Fiscal Regimes.

To create a formula for a variable

  1. In the Fiscal Regimes Editor, select the variable you want to add a formula to.
  2. Under Formula, click .
  3. In the Formula Editor, move formula elements to the Formula window.
    4. Click a formula element to see a description of it at the bottom of the Formula Editor.
  4. Click OK.
