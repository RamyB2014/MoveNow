Group Fields in Custom Table Reports

Grouping fields in a report makes the report easier to read by separating the results into groups. For example, you can group the data by Date (day, week, month, quarter, or year). Adding groups to a report also gives you the ability to display subtotals.

To add a group field to a report object: In the Report Editor, drag fields from the Fields and Formulas window and drop them into the Groups window.

The fields that you drop in the Group Fields window also appear in the Sort Fields window, where they can be assigned an ascending or descending sort order.

Group Fields can also be summarized or nested. See Summarize Grouped Fields in Custom Table Reports and Nest Group Fields in Custom Graph Reports.
