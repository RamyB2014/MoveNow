Remove the Password from a Project Connection File

By default, project connection files (.vndl) for Oracle and SQL Server projects contain the password (encrypted). This means that any user who opens the file can access the project without entering a password. If you want to make the file accessible to other users but require them to enter the password you give them, you can remove the password from the connection file.

To remove the password from a connection file

  1. Launch and close the Project Launcher. If you have a project open, click  to close the project ( remains open).
  2. From the File menu, select Edit or Create a Project Connection File.
  3. In the File Path, click.
  4. Browse to the .vndl file you want to edit and click Open.
  5. Select Exclude password from file.
  6. Click Save and Close.
