Edit Exchange Rates

To edit exchange rates

  1. In the Tools menu, select Global Project Data > Price Decks.
  2. In the Price Deck Editor, click Exchange Rates (on the left).
  3. Select a Reference Currency.
  4. Edit the currency values relative to the reference currency.
  5. Click Save.

To add a currency, click Edit Currencies. You can add a new currency here, but you cannot add that currency to a country. To add the currency to a country, see Set the Country Currency.
