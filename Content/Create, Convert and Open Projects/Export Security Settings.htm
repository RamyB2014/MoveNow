Export Security Settings

You can export Security Settings for use in another project or for sharing with other users. You can export Security Settings from the Export to File dialog box (accessed by selecting Export from the File menu) or you can export them from the Security Settings dialog box (accessed by selecting Security Settings from the Administration menu). When you export Security Settings from the Security Settings dialog box, the export file only contains Security Settings. However, if you export them from the Export to File dialog box, you can include other project data (see Export Project Data for more information).

To Export Security settings (from the Security Settings dialog box)

  1. In the Administration menu, select Security Settings.
  2. In the Security Settings dialog box, click Export Security.
  3. Browse to a location to save the file.
  4. Type a file name and click Save.
