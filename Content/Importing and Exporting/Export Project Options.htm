Export Project Options

You can export Project Options (and project data) for use in other projects to ensure consistent configuration.

You can export project options only (all options on the Project Options tab of the Options dialog box) using a ValNavProjectOptions file or you can include project data by creating an xml file.

Export Project Options (using a ValNavProjectOptions file)

When you export Project Options from the Options dialog box, the export file only contains Project Options.

To export project options (from the Options dialog box)

  1. On the Tools menu, click Options.
  2. In the Options dialog box, click the Project Options tab.
  3. Click Export.
  4. Browse to a location accessible to other users who need to use the file.
  5. Enter a name for the file and click Save.

Export Project Options and Project Data (using an xml file)

To create a project options file that includes other project data, see Export Project Data.
