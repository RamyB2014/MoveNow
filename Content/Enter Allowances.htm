Enter Allowances

When you enter Gas Cost Allowance and Custom Processing Fee data, both sets of data are used unless you select Do not calculate operating or capital GCA (see the procedures below).

When you select a Meter Station, you can edit the factor for the selected well or edit the factor globally in the Meter Station Editor.

To enter allowances for Alberta

  1. Go to Economics | Prices & Costs | Allowances.
  2. Under Transportation Allowance, select the Area from the Area list.
    Values for C3, C4, C5+, and NGL are displayed in the Values window. Edit the values, if required.
  3. Under Royalty Allowance, enter Fractionation and Storage values.
  4. Select a Meter Station.
  5. Enter values for Gas Cost Allowance and/or Custom Processing Fee.
  6. Select Apply allowances to freehold and ORR if you have freehold land and/or overriding royalty where GCA is applied.
  7. Select Do not calculate operating or capital GCA if you want to ignore GCA entries.

To enter allowances for British Columbia

  1. Go to Economics | Prices & Costs | Allowances.
  2. Enter values for Producer Cost of Service.
  3. Enter values for Gas Cost Allowance and/or Custom Processing Fee.
  4. Select Apply allowances to freehold and ORR if you have freehold land and/or overriding royalty where GCA is applied.
  5. Select Do not calculate operating or capital GCA if you want to ignore GCA entries.
