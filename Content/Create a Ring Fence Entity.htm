Create a Ring Fence Entity

You can create a ring fence entity (RFE) from selected entities or you can create an empty ring fence and add entities to it.

To create an RFE from a selection of entities

  1. In the Entity explorer, select the entities you want to include in the RFE.
  2. Right-click the selected entities, point to Create As, and select Ring Fence.
  3. In the Create Ring Fence dialog box, enter the Entity Properties and Custom Fields, if required.
  4. Select the Fiscal Regime you want the RFE to use.
  5. For ring fences with wells that are eligible for the Emerging Resources Program in Alberta, select Emerging Resources Program in the Calculation list.
  6. Click OK.
  7. On the Ring Fence tab, select one of the two Child results options:
    • Honour Economic Limit on child wells
    • Ignore Economic Limit on child wells
  1. Select or deselect Apply termination date to children.

Also see Create a Fiscal Regime.

To create an empty RFE and add entities to it

  1. From the Entity menu, point to Create and select Empty Ring Fence.
  2. In the Create Ring Fence dialog box, enter the Entity Properties and Custom Fields, if required.
  3. Select the fiscal regime you want the RFE to use.
  4. Click OK.
  5. In the Explorer, select the entities you want to add to the RFE.
  6. Right-click the selected entities, point to Add to, and select Ring Fence.
  7. The Add to Entity dialog box, select the RFE to which you are adding the selected entities.
  8. Click OK.
  9. On the Ring Fence tab, select one of the two Child results options:
    • Honour Economic Limit on child wells
    • Ignore Economic Limit on child wells
  1. Select or deselect Apply termination date to children.
