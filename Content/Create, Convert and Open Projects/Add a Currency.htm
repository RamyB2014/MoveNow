Add a Currency

comes with three currencies already created: US dollar, Canadian dollar, and the Euro. Use the following procedure to add more currencies.

To add a currency

  1. On the Tools menu, select Global Project Data and click Countries & Currencies.
  2. In the Countries & Currencies Editor, click the Currencies tab.
  3. Under the Currency list, click Add.
  4. In the Currency Name dialog box, enter a name and click OK.
  5. Select the new currency from the currency list.
  6. Edit the Currency Properties, as required.

To assign a currency to a country, see Set the Country Currency.
