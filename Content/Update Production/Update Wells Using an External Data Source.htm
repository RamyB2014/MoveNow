Update Wells Using an External Data Source

This procedure explains how to update monthly or daily well production using PPDM (IHS or GeoLogic) or a corporate data hub. Using this method updates the currently selected wells.

If you want to update a subset of wells in your project, filter to those wells first. See Flag and Filter Entities.

If you want to schedule an update for a later time, or update a subset of wells based on a well list, see Schedule a Production Update.

See Production Update Overview for more information about updating wells.

To update production

  1. On the Entity menu, click Update from External Source.
  2. Under Entity Selection, select the wells you want to update.
  3. Select your Data Source and click OK.
