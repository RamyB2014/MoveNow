Add a Constraint to a Schedule

In Schedules, a Constraint is a line displayed on the Schedules timeline graph that represents a limit which can be economic (cumulative capital) or technical (volume or rate). Displaying a Constraint can help make it clear when your entity or schedule exceeds an economic or technical limit.

To add a constraint

  1. On the Timeline tab, click Add Constraint (top right).
    A new constraint is added.
  2. From the Constraint list, select a constraint.
  3. Enter a value for the Constraint. The Constraint is displayed on the Timeline graph.
