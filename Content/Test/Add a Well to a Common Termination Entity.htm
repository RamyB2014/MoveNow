Add a Well to a Common Termination Entity

To add a well to a common termination entity (CTE)

  1. In the Explorer, select the well you want to add to a CTE.
  2. Right-click the well, point to Add to, and click Common Termination Entity.
  3. In the Add to Entity dialog box, select the CTE and click OK.

To view CTEs in the Entity Explorer, select Common Termination from the Entity Type list in the Standard toolbar.

If a well you are adding to the CTE is already in another CTE or group, a message will ask you if you want to remove it in order to move it into the CTE. The well cannot exist in more than one CTE or group.
