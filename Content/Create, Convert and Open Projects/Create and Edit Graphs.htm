Create and Edit Graphs

You can create and customize graphs in three ways:

Method

Description
Create a new graph A new graph is one you build from scratch and is not based on an existing graph. See Create a New Graph.
Create a graph version A graph version is based on a source graph and remains related to it. Graph versions are not displayed in the graphs list—only the source graph is displayed. If you select the source graph, any related graph versions are automatically used if its criteria match the Entity. See Create a Graph Version.
Create a graph copy All default graphs are read-only, but you can make a copy of a default graph, edit it, and save it with a new name. Unlike a graph version, graph copies are displayed in the graphs list and not related to their source graph. You can create graph versions of graph copies. See Create a Graph Copy.
