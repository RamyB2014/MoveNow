Apply Workspace Forecasts

Using Apply Workspace Forecasts, you can apply a workspace forecast to a plan and reserves category for several wells at once. Also see Create and Manage Forecast Names.

To apply workspace forecasts

  1. From the Tools menu, select Apply Workspace Forecasts.
  2. At the top of the dialog box, select the Reserves Category and Forecast Name of the forecast you want to apply.
  3. Under Destination, select the destination Plan and Reserves Category.
  4. Select the options below, as required:
    5. Option Description
    Apply to children of groups Applies the forecast to children in a group.
    Create a copy of the target active forecast. Creates a copy of a well’s current forecast.
  5. Click Apply.
