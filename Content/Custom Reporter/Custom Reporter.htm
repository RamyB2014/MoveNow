Custom Reporter

The Custom Reporter can be used to create Table or Graph reports, on Economic or Technical output. Reports can be based on filters and consist of several predefined objects resulting in reports that only display output from certain reserves categories, statuses, or scenarios.

Working with Custom Reports

When creating custom reports, keep the following points in mind:

  • Always run Economics before running an economic custom report to ensure the economic data is current.
  • When creating custom reports, always add filters to limit the reserves categories and reserves statuses. See Add Filters to Custom Table Reports or Add Filters to Custom Graph Reports.
