Create Users

Use this procedure to create a single user and assign that user a password.

To assign the user a role or policies, see Assign Roles to Users or Assign Policies to Users.

To import users from an active directory, see Import Users from an Active Directory.

To create a user

  1. From the Administration menu, select Security Settings.
  2. On the Users tab, click Create User.
  3. Enter a User Name and Password and click OK.
