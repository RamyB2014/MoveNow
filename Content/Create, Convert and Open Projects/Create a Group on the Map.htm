Create a Group on the Map

To create a map group

  1. Select the entities you want to group together by doing one of the following:
    • Ctrl + click the entities in the Entity Hierarchy, or
    • Ctrl + click the entities on the map
  1. Right-click in the Entity Hierarchy and select Create as Group to open the Group Setup dialog box.
  2. Enter a Group Name and click OK. The group is indicated on the map by a derrick icon.

To remove an entity from a map group

  1. In the Entity Hierarchy, select the well that you wish to remove from the group.
  2. Right click and select Remove from Group.
    A dialog appears confirming that you wish to remove an entity from the group.
  3. Click Yes to remove the entity.

To delete a map group

  1. Select the group in the Entity Hierarchy.
  2. Right-click and select Delete Entity. The Delete Entity dialog appears, reminding you that the Group will be removed from the project and the child entities will be unlinked.
  3. Click Yes to delete the group.
