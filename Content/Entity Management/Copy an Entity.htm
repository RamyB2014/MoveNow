Copy an Entity

The Copy Entity feature copies all data associated with an entity. This includes inputs in all existing plans and copying Change Records associated with a Posted database.

To copy a well

  1. In the Entity Explorer, right-click the well you want to copy and click Copy Entity.
  2. Type a Well ID.
  3. Select the well data you want to copy to the new well.
  4. If you want to adjust the well’s dates, click Adjust Dates. See Adjust Entity Dates.
  5. Click OK.
