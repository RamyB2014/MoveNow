Work Offline

If you need to use offline, you need to request a license for offline use while you are still connected to the internet.

Your administrator configures to allow a specified number of licenses for offline use because when you work offline, the license you are using is not available to other users. The administrator also configures the maximum number of days you can use the license offline.

To request a license to work offline

  1. From the Help menu, select License Manager.
  2. In the License Manager, click Work Offline.
  3. Select the date you require the license until.
    4. Only available dates are displayed.
  4. Click OK, and Close.
