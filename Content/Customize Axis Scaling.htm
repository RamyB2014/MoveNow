Customize Axis Scaling

You can customize axis scaling for graphs on Predictions | Declines, P/Z, and Volumetrics.

To customize axis scaling

  1. Right-click any graph on Predictions | Declines, P/Z, and Volumetrics and select Graph and Scaling Options.
  2. Select Use Custom Scaling.
  3. Do one of the following:
    4. Select And
    Custom X Axis Scaling, In the Scaling Mode column, select Manual for the Products you want to customize. Enter a Minimum and Maximum value.
    Custom Y Axis Scaling, For each item, enter a maximum value and select Use Custom Scaling. OR Click Calculate From Well.
  4. Click OK.
