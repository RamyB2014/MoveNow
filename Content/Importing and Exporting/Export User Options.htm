Export User Options

Your user options are always applied to the project you have open. However, you can export them for sharing with other users.

User Options include all items on the User Options tab of the Options dialog box.

To export user options

  • On the Tools menu, click Options.
  • In the Options dialog box, click the User Options tab.
  • Click Export.
  • Browse to a location for the file.
  • Enter a name for the file and click Save.
