Open a Project as a Copy

You can only open project files (.vndb) as a copy. You cannot open server-based files (Oracle and SQL Server) as a copy.

To open a project as a copy

  1. Launch or click  (open) if you have it open already.
  2. In the Project Launcher, click Browse.
    3. If you select a file from your recently-opened list, you cannot open it as a copy. To open a file as a copy, browse to the file instead.
  3. In the Open Project dialog box, browse to the project file you want to open as a copy.
  4. Copy the project file and paste it into the folder (or a different folder) with a new name.
  5. Double-click the copied file to open it in .
