Generate the Incremental Production Variance Report

The Incremental Production Variance Report displays the incremental production between a baseline scenario and another scenario, archive, plan, or reserves category.

To generate the Incremental Production Variance Report

  1. From the Tools menu, point to Incremental Reports and select Incremental Production Variance Report.
  2. Complete the following options:
    3. Option Description
    Start Year Only wells with a production start date in the year specified are included in the report. The report cannot start before the Economic Calculation Start Date.
    Months to display Number of months in the Start Year to be displayed (starting with January of the Start Year).
    Maximum hierarchy depth The number of folder levels to report on.
    Baseline scenario The scenario to serve as a baseline for the incremental report.
    Minimum Variance The minimum amount of variance that will be reported. Variance is not reported below this value.
    Archive Archive for comparison to the Baseline.
    Plan Plan for comparison to the Baseline.
    Reserves Category Reserves Category for comparison to the Baseline.
    Scenario Scenario for comparison to the Baseline.
  3. Click OK.
  4. In the Print Preview, select an entity or folder level to view the results.

If you do not select Preview on Selection in the bottom left of the Print Preview, report results are not displayed when you select an entity or folder. To display the results, click Preview.

If you select Preview on Selection, the report results are automatically displayed when you select an entity or folder.
