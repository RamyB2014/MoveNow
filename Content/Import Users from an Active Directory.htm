Import Users from an Active Directory

Importing users from an active directory enables users to log into projects without entering an additional user name or password if they are already logged into the network.

To import users from and active directory

  1. From the Administration menu, select Security Settings.
  2. In the Security Settings dialog box, click Active Directory Settings.
  3. In the Active Directory Settings dialog box, select Use Active Directory Integration.
  4. From the Active Directory Group list, select the group you want to use and click OK.
  5. In the Security Settings dialog box, click Import AD Users.
  6. In the Import Active Directory Users dialog box, select the users to import and click OK.
