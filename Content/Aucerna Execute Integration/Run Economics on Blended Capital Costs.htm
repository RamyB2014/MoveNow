Run Economics on Blended Capital Costs

In you can run economics on forecasted capital costs or blended capital costs. Blended capital costs are a mix of forecasted and actual costs. You can only run economics on blended capital costs if you have imported actual capital costs from AFENav or another source.

You can run economics on blended capital costs by selecting Run Economics on Blended Costs in the Project Options, under Economics. Economics are run on forecasted capital costs by default (Run economics on forecast only option).

You can view forecasted and actual capital costs on Economics | Prices & Costs | Capital Costs. To see the forecasted and actual costs, select Forecast and Actuals in the View list. The green column displays actuals.

The suggested forecast is displayed below the End of Actuals line. If the suggested forecast is negative, you have spent more than you forecasted.

You can filter to overspent entities by creating a filter with Has Capital Overspend= True. See Create a Filter Query.
