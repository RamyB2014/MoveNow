Map API Numbers to Wells

You can map API numbers to wells in two ways:

To map a large number of API numbers to wells, use the Spreadsheet Import tool.

To map a single API number to a well, use the API Numbers dialog box in the Administration menu.

To change a well name to the API number, see Rename an Entity.

An entity’s API is displayed in the Properties Window (under Identification) in the bottom left of the screen when you select the Properties tab. The API is also displayed on Well and General Economics in the Data view.

Using the Spreadsheet Import Tool

To use the Import Spreadsheet, see Spreadsheet Import.

If you import more than one API number for a well, only the first API number is displayed on Wells and General Economics. To see other mapped API numbers, select API Numbers from the Administration menu. In the API Number dialog box, you can change the Primary API Number by right-clicking a number and selecting Set Primary API.

Using the API Number Dialog Box

To map an API number to a well

  1. In the Explorer, select one or several entities.
  2. From the Administration menu, select API Numbers.
  3. In the Primary API Number column, enter the API number to map to each UWI. Enter additional API numbers in the other API Number columns.
  4. Click OK.

Go to Wells and General Economics to see the API Number column.

If you enter more than one API number for a well, only the first API number (Primary) is displayed on Wells and General Economics. In the API Number dialog box, you can change the Primary API Number by clicking a number and selecting Set Primary API.
