Edit an Archive

Editing archives always involves removing information. Once you have edited an archive, you cannot add the deleted information back again. If you are editing an archive, it is recommended that you save the edited version with a new name.

If you cancel the editing of an archive (if you click Cancel in the Edit Archive Progress dialog box) the newly-created archive will still be displayed as a Full Archive even though some data has been removed. Note that this new “Full Archive” will be missing data. See the procedure below for more detail.

To edit an archive

  1. From the Administration menu, select Archive Manager.
  2. In the Archive Manager, click the archive you want to edit (highlight it), then click Edit.
  3. In the Edit Archive dialog box, under Archive Plans, select the plans you want to include in the archive.
  4. Under Archive Scenarios, select the scenarios you want to include.
  5. Enter a name and description for the archive and select the following options as required:
    6. Option Sub-option Description
    Partial Archive Technical Data: Declines Entities: Wells, groups, and type wells. Data: Forecast parameters, manual overrides (Data tab), and auto fit method.
    Technical Data: Declines and other inputs: Entities: Wells, groups, and type wells. Data: Forecast parameters, manual overrides (Data tab), volumetrics, P/Z, Plant tab inputs and overrides, gas analysis inputs and overrides, type well link, and auto fit method.
    Economic Data:  Results Entities: Wells, groups, and common termination entities. Data: All data in output tables (including after-tax results, but not break even results).
    Economic Data: Results and Basic Inputs Entities: Wells, groups, and common termination entities. Data: All data in output tables (including after-tax results, but not break even results), capital cost inputs, operating cost inputs, and interest inputs.
  6. Click OK.

Once you click OK, the Edit Archive Progress dialog box displays the archive progress. Clicking Cancel at this point will result in an archive displayed as “Full”. However, the archive will be missing data.
