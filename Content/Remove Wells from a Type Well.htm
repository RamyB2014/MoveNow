Remove Wells from a Type Well

There are two methods for removing wells: Remove and Remove Excluded (see Exclude a Well from a Type Well).

To remove a well from a type well

  1. From the Entity Type list in the main toolbar, select Type Wells.
  2. In the Explorer, select the well(s) you want to remove from the Type Well.
  3. Right click and click Remove from Type Well.

To remove wells you have already excluded, click Remove Excluded on the Type Wells tab.
