Apply a Layout Template to a Tab in Data View

To make configuration of a tab quicker, you can apply a layout template, which may have been supplied to you by an Administrator or another user.

Tab layout determines how the data in the grid is displayed. Layouts determine which fields are displayed in the rows and columns of the grid, which data are shown in the grid (as [Values]), and which filters are applied to the data in the grid and within each field.

To apply a layout template to a tab in the Data View

  1. Select the appropriate tab and click  to display the layout panel.
  2. At the top of the layout panel, click Apply Template.
  3. Select the template you want to apply.
    The template is applied to the current tab.

To restore the default layout

  1. In the Layout Panel, click Apply Template.
  2. Click Default.
