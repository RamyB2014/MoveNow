Generate a Spreadsheet Import Template

You can use the Spreadsheet Import tool to import data from your existing spreadsheets (.xlsx, .xlsm, or .csv files) without having to copy data or create new spreadsheets.

However, if you are unsure what type of data you can import or are unsure of how to format the data, you can generate a spreadsheet template for a mapping area. When you generate a template it creates a spreadsheet that contains all possible import columns for the mapping you specify. In the spreadsheet template, each column header has a comment that explains the format of the data to be entered.

To generate a spreadsheet import template

  1. From the File menu, select Spreadsheet Import.
  2. Select a Mapping from the list and click .
  3. In the Save As dialog box, type a file name (if required) and click Save.

An Excel spreadsheet opens with column headings configured according to the Mapping you selected. Place your mouse pointer over the red comment tag in each header to view an explanation of the data requirements and formatting.
