Customize Predictions | Declines Fields

By default, the Predictions | Declines tab displays the Field, Pool, On-time, and Type Well Link fields above the decline parameters. However, you can change the fields that are displayed or add more fields.

To add or edit the fields displayed on Predictions | Declines

  1. On Predictions | Declines, click .

  2. Under Quick Display Fields, do one of the following:
    3. To Do this
    Change a field, Click the field name and select a new field.
    Move a field to a different location, Select the field and click the Up or Down arrow.
    Add a new field, Click Add and select a new field name.
    Remove a field, Click the field and click Remove.
    Hide or show the field label, Select or deselect Show Label.
  3. Click outside the dialog box to exit. Your changes are automatically saved.
