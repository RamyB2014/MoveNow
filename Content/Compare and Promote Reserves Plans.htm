Compare and Promote Reserves Plans

The Compare and Promote function allows you to generate a report that compares reserves for each entity in different reserves categories and different reserves plans. It also allows you to promote multiple entities from one plan to another.

Before promoting the entities, you can generate a report for a selected set of wells, for a selected set of reserves categories comparing the Source and Destination reserves plans.

To generate the Compare Reserves Plan report

  1. From the Change Management menu, select Compare/Promote.
  2. On the Report tab of the Compare/Promote Reserves Entities dialog box, select the Source and Destination reserves plans you want to compare.
  3. In the Entity Hierarchy, select the folder levels or entities you want to compare.
  4. On the Report tab, select the reserves categories you want to compare.
  5. Select one of the following:
    6. Option Description
    Show only selected wells with differences Only displays wells with differences between the Source and Destination reserves plans in any reserves category
    Show all selected wells Displays all selected wells regardless of whether there are differences between reserves plans
  6. Click Generate Report.

The report is displayed with values for the Source and Destination reserves plans and the difference between them.

To promote entities from one reserves plan to another

  1. On the Change Management menu, click Compare/Promote.
  2. Select the Promote tab.
  3. In the Entity Hierarchy, select the folder levels or entities you want to promote.
  4. Select the Source and Destination reserves plans you are promoting.
  5. Click Copy.
