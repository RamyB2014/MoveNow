Project Options: External Project Data

Creating an External Project Data file and then applying it to other projects enables you to create projects with identical project data (as listed under External Project Data). If the project data changes, you can update the external project data file to reflect the changes.

To create an external project data file

  1. Open a project with the project data you want to apply to other projects.
  2. From the Tools menu, select Options.
  3. In the Options dialog box, select the Project Options tab and then select External Project Data.
  4. Specify which data you want to include in the file and click Create External File.
  5. In the Select External Project Data File dialog box, browse to a location and click Open (to save the file).
Once you create a project file, the Create External File button name changes to Update External File.

To apply an external project data file to a project

  1. Open the project to which you want to apply the external project data.
  2. From the Tools menu, select Options.
  3. In the Options dialog box, select the Project Options tab and then select External Project Data.
  4. Click  next to External project data file and browse to the file.
  5. In the Select External Project Data File dialog box, browse to the file and click Open.

To update an external project data file

  1. Open the project containing the project data file you want to update.
  2. From the Tools menu, select Options.
  3. In the Options dialog box, select the Project Options tab and then select External Project Data.
  4. If required, change your selection of options under External project data selection.
  5. Click Update External File.

External Project Data File

The specified xml file populates the current project with an external data set. If you do not have a file to browse to, you can create one using the Create External File button.  

External Project Data Selection

Select the project data that you want to update in the existing external project file, or use to create a new one.

Create/Update External File

When the External project data file is specified, the Update External File button is enabled.

  • When the External project data file is not specified, the Create External File button is enabled.
