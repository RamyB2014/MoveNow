Set the Default Country

When you set a country as the default, the Country field is automatically populated when you create a well. If you import a well from another project, the well’s country is retained.

To set the default country

  1. On the Tools menu, click Options.
  2. In the Options dialog box, click the Project Options tab.
  3. On the left, select General.
  4. On the right, select the country from the Default Country list and click OK.
