Create a Well Using a Type Well

Type Wells display an averaged representation of the normalized production history (and forecast, if selected) for a set of wells. However, this representation is not a decline so you must create a decline for the Type Well before you can create a new well based on it.

The procedure below explains how to create a decline for a Type Well and then create a new well with that decline.

To create a well from a Type Well

  1. From the Entity Type list in the main toolbar, select Type Wells.
  2. Select a Type Well and go to Predictions | Declines.
    3. The Type Well’s normalized data is displayed. This is not a decline.
  3. Create a decline by drawing a profile.
  4. Go back to Type Well | Definition and click Create Well.
  5. In the Well Information dialog box, complete the following options:
