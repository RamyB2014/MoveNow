To import a spreadsheet

  1. From the File menu, point to Import and select Spreadsheet Import.
  2. Do one of the following:
    3. To Do this
    Generate a spreadsheet template, Click next to Mapping.
    Use your own spreadsheet, Next to File, click  to browse to the spreadsheet you want to import, and click Open.
    The content of the spreadsheet is displayed in the Spreadsheet Import tool.
  3. If required, select a sheet (.xlsx files only) from the list next to the File selection field.
  4. From the Mapping list, select a data mapping.
  5. In Header rows (below the preview), enter the number of header rows to exclude from the import. Header rows are rows that you do not want to import.

    6. In this import, the first 3 rows are not part of the import.

  6. Review the column heading configurations to confirm they are correct.
    7. You can assign a single value for Required Fields by setting the corresponding column mapping to None and then selected a value in the Required Fields area.

    In this import, the Plan, Reserves Category, Company, Lease, Product Interest, and Reversion columns have been set to None and assigned a single value in the Required Fields area.

  7. Unmap any empty columns by highlighting the column headers and pressing the Delete key (which sets the column heading to <None>).
    8. Importing empty columns or rows for fields that already have data in your project will overwrite the existing data with a null value.
  8. If you need to edit the content of the spreadsheet before importing:
  • Click  (next to Sheet) to open the source file.
  • Click  (below the preview) and select one of the following Operations: Merge, Replace, or Replace All.
    Before selecting an Operation, carefully read the tool tip for each operation by placing your mouse pointer over the Operation.
    The Operation options control how imported data will affect existing data. The Operations behavior depends on the data mapping type you are working with.
  1. Click Validate(bottom right). If there are validation errors, click Open log (next to Validate) to view the log file. Correct any errors and continue.
  2. If required, click Add Task to add another task and repeat the steps above to configure the task.
  3. Click Import (to import a single task) or Import All (to import all tasks).
