Keyboard Shortcuts

Keyboard shortcut keys are listed below.

Key Area in Application Function
F1 Anywhere Opens Help file
F3 Anywhere Opens Find dialog box or Finds next after using Find
F4 Anywhere Opens Entity Properties
F5 Anywhere Refreshes the Explorer
F6 Anywhere Opens Entity Explorer
F7 Anywhere Opens Entity Summary
F11 Anywhere Enters full screen mode
Alt + Right Arrow Anywhere Move forward in the navigation history.
Alt + Left Arrow Anywhere Move back in the navigation history.
Ctrl + B Predictions |Declines Best fits a decline from a selection
Ctrl + R Predictions |Declines Refits a decline from a selection
Ctrl + F Anywhere Opens Find dialog box
Ctrl + P Anywhere Runs economics and opens Print Preview window for reports
Ctrl + T Predictions |Declines, Status, Volumetrics Applies Data Point Tracking to graphs
Alt + Left Click Predictions | Declines Best fits a decline from a point
Ctrl + Left Click Predictions | Declines Refits a decline from a point
