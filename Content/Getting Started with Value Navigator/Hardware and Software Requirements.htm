We have encountered an issue when running on Oracle 12C, version 12.1.0.1. An Oracle 12C feature called Adaptive Plans has a bug that causes bad data to be sent back to , which can result in a crash. We have seen these crashes only on upgrades but cannot rule out their occurrence during regular operations. If you plan to use Oracle 12C, we recommend you run version 12.1.0.2 as we have not encountered this error with this version.
