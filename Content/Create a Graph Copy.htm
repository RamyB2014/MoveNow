Create a Graph Copy

The default graphs in are read-only graphs. However, you can make a copy of a graph and then edit it in the Graph Designer.

A graph copy is a “standalone” graph that is different than a graph “version” (versions are related to their original graph). See Create a Graph Version.

To create and edit a copy of a graph

  1. On Predictions | Declines or Predictions | Volumetrics, right-click any graph and click Select/Manage Graphs.
  2. In the Select/Manage Graphs dialog box, click the report you want to make a copy of and click Edit.
    The Graph Version Manager is displayed.
  3. Select the graph version you want to copy and click Edit. 
    “This graph is read-only. Would you like to edit a copy” is displayed.
  4. Click Yes.
    The Graph Designer is displayed.

See Edit Graphs for an explanation of how to build the graph.
