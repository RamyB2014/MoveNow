Edit Inflation

There are three types of inflation in : Price Differentials, Capital Cost, and Operating Cost.

Capital Cost and Operating Cost inflation are set to 2% by default. Price Differentials inflation is set to 0% by default. See Inflation of Price Differentials (Offsets).

To edit inflation

  1. In the Tools menu, select Global Project Data > Price Decks.
  2. In the Price Deck Editor, click Inflation (on the left).
  3. In the editor window, open the General folder.
  4. Edit the appropriate inflation types in the date grid on the right.
  5. Click OK.
You can prevent Capital Cost and Operating Cost inflation for each cost by selecting Prevent Inflation for each cost on Economics | Prices and Costs. See Enter Operating Costs or Enter Capital Costs.
