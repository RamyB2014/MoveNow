Pin the Explorer or Summary/Properties Window

Pinning the Explorer or the Summary/Properties windows:

  • Maximizes the window when it’s in use
  • Auto hides the window when it’s not in use to create more space on the screen
  • Displays a tab on the left side of the screen for each pinned window

To pin a window, at the top of the window, click . The window will be maximized. The window is hidden when you are not using it.

To display a hidden window, put your mouse pointer over its tab.

To unpin a window, click .

To unpin all windows and restore the default display, select Restore Default Layout from the View menu.
