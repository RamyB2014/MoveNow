Add a Product to an Entity

To add a product to an entity, you must add the product to a product list first. You can then specify that the entity use that product list.

To add a product to an entity

  1. If required, create the product and add it to a product list.
  2. In the Entity Explorer, right-click the entity you want to add the product to and click Edit Entity.
  3. Select the Product List that contains the product you want to add and click OK.

The product will be displayed on the relevant tabs.

If you want to forecast a custom product, you must select Allow Decline when you create the product. See Create a Custom Product.
