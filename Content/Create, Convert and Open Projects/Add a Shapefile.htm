Add a Shapefile

You can customize the map display by importing shapefiles. You can add shapefile layers that show time zones, latitudes and longitudes, states and countries, etc. Shapefiles are saved for future use per user, not per project.

  • Current Shapefile Layers: Displays the layer files that have already been added. The Move Up and Move Down buttons are used to change the order of the layers and specify the top layer in the map.
  • Layer Properties: Displays the name of the layer currently in use, what type of layer it is (Areas, Lines, etc.) and the location where the shapefile is saved.
  • Line colour, Fill colour and Symbol: Dropdown lists used to customize the appearance of the layer.
A shapefile can be used instead of the default base maps. Remove the default base map by selecting No Base Map from the Base Map list under System Layers and then assign a shapefile layer to the map.

Shapefiles are made up of three files with the extensions .shp, .shx and .dbf. If any of the files are missing, an error message will be displayed and the layer will appear as “Missing” under Shapefile Layers.

To add a shapefile

  1. On the Map tab, click .
  2. Click Add shapefile.
  3. Browse to the location of the shapefile you want to use and click Open.
  4. Specify the layer options, such as Line colour, Fill colour, Symbol, or Transparency.
  5. Specify the following options, as required:
    6. Option Description
    Dynamic visibility scale range The map scale range in which the layer is visible. Enter the scale with out “1:”. For example, enter “6900000” not “1:6900000”.
    Label The map scale range in which the labels are visible. Enter the scale with out “1:”. For example, enter “6900000” not “1:6900000”. If you don’t enter Min and Max values for the Label, the Dynamic visibility range values are used automatically.
