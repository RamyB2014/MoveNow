User Options: File Locations

Report and Graph Definitions

Local Files Directory: This is the location for local (personal) reports and graphs. Custom reports and graphs are saved in a Reports or Graphs folder saved in this location. The default location is C:\.

Shared Files Directory: This is the location for shared reports and graphs. Custom reports and graphs are saved in a Reports or Graphs folder saved in this location. The default location is the installation directory. You must specify a shared drive to share reports.

When you save your first report or graph, creates a folder named Reports or Graphs in the Local Files or Shared Files Directory. These folders are not visible from the interface when you save reports or graphs, as reports are automatically saved in the reports folder and graphs are automatically saved in the graphs folder. If you change the Local Files or Shared Files Directories, you must move these Reports and Graphs folders to the new locations in order to see the reports or graphs you previously created.

Project Default Directories

Data Directory: This is the default location for browsing to a source file from the Project Launcher (when you select Create from Import).

Project Directory: This is the default location for saving or opening projects in the Project Launcher (when you select Create New Project or Browse).

Default Project Data File

To ensure projects have consistently configured Project Options and/or project data you can select a ValNavProjectOptions file or an .xml file that contains settings that are applied to all new projects you create.

Note that selecting a default project data file does not apply its settings to your current or past projects. The file is only applied to projects you create after specifying the file. If you want to apply external project settings to your current project, see Project Options: External Project Data.

Once you specify a default project data file, the file path is displayed in the Default Project Data File field for all projects (current, past, and new), but its settings are only applied to projects created since the file was specified.

To create a default project data file, see Export Project Options.
