Use a Type Well to Create Related Type Wells

You can create copies of a type well and assign those copies different factor types, such as P Mean, P10, or a custom factor percentage.

To create related type wells

  1. From the Entity Type list in the main toolbar, select Type Wells.
  2. Select the type well to edit.
  3. Go to the Type Well tab, click Copy To, and select a copy type.
  4. Type a Name for the type well and/or edit the Percentile type and value.
  5. Click OK.

Also see Create a Type Well and Edit Type Well Options.
