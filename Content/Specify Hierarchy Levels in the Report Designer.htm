Specify Hierarchy Levels in the Report Designer

The hierarchy depth controls how many levels of the hierarchy are displayed in hierarchy reports. The Unlimited setting displays all folder levels and entities.

To specify hierarchy levels in a Report Designer report

  1. Create a Report in the Report Designer or open a report you already created.
  2. Click the hierachy table you want to add levels to.
  3. Under Properties (bottom left), change the Hierarchy Depth.
