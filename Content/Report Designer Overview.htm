Report Designer Overview

You can create and customize report designer reports in three ways:

Method Description
Create a new report A new report is one you build from scratch and is not based on an existing report. See Create a New Report.
Create a report version A report version is based on a source report and remains related to it. Report versions are not displayed in the reports list—only the source report is displayed. If you select the source report, any related report versions are automatically used if its criteria match the entity. See Create a Report Version.
Create a report copy You can make a copy of a report, edit it, and save it with a new name. Unlike a report version, report copies are displayed in the reports list and not related to their source report. See Create a Report Copy.

You can also create and customize economic and technical reports in the Custom Reporter.
