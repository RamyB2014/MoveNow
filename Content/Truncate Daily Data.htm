Truncate Daily Data

Truncate Daily Data removes daily production data prior to the date you specify.

To truncate daily data

  1. In the Administration menu, select Data Cleanup > Truncate Daily Data.
  2. Select a date. All daily production data with a date prior to this date will be permanently deleted.
  3. If you want to preserve production data from the start of production, specify the number or days in Days to preserve from the start of production.
  4. Select Apply to Children of Groups if you want to remove daily production data from child wells as well.
  5. Click OK.
