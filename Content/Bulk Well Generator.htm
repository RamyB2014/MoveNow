Bulk Well Generator

The Bulk Well Generator (BWG) enables you to create a well schedule using a source well as a starting point for creating more wells that will be drilled according to a schedule. You can specify the:

  • Number of wells to be drilled
  • Number of rigs available for drilling
  • The start date of each well
  • A common production date for all wells
  • COS and COO factors for each well
  • Production and Capital factors for each well

You can also:

  • Update all wells in the schedule if you make changes to the schedule’s source well
  • Adjust the start date of the schedule or specific wells in the schedule
