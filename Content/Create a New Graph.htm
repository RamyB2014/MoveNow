Create a New Graph

A new graph is one you build from scratch and is not based on an existing graph. To use a default graph as the source of a new graph, see Create a Graph Copy.

To create a new graph

  1. On Predictions | Declines or Predictions | Volumetrics, right-click any graph and click Select/Manage Graphs.
  2. In the Select/Manage Graphs dialog box, click New. The Graph Designer is displayed.

See Edit Graphs for an explanation of how to build the graph.
