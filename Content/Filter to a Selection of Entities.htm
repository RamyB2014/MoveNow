Filter to a Selection of Entities

This procedures below explain how to filter your project to the entities you have selected in the Entity Explorer or in the Data View grid.

See Using the Data View.  

In Entity View

To filter your project to selected entities in Entity View

  1. In the Entity Explorer, select the entities you want to filter to.
  2. Right-click in the hierarchy and select Filter to Selection.
When a filter is applied to the project, the Filter Well List icon is red .

To clear the selection filter

  1. On the Standard toolbar click  and click Clear Selection Filter.

In Data View

To filter your project to selected entities in Data View

  1. Right-click an entity in the Data View grid.
  2. Select Filter Hierarchy to Selection.
When a filter is applied to the project, the Filter Well List icon is red .

To clear the selection filter

  1. On the Standard toolbar click  and click Clear Selection Filter.
