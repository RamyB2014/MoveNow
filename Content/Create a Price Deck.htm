Create a Price Deck

To create a new price deck, you have two options: create one from scratch or create a copy of a price deck that is similar to the new one you want to create. Creating a copy is significantly easier because you don’t need to edit all of the price deck attributes (such as royalties and tax rates). 

To copy a price deck

  1. In the Tools menu, select Global Project Data > Price Decks.
  2. Next to Select Price Deck, select the price deck you want to copy.
  3. To the right of the Select Price Deck list, click Copy.
  4. Enter a name for the new price deck and click OK.
  5. As required, edit the stream prices, exchange rates and create price sets as required.
  6. To the right of the Select Price Deck list, click Make Active to apply the price deck to your project.
  7. Click OK in the Price Deck Editor.

To create a new price deck from scratch

  1. In the Tools menu, point to Global Project Data and click Price Decks.
  2. In the Price Deck Editor, click New Price Deck.
  3. Type a name and click OK.
  4. Edit the stream prices, Edit Exchange Rates, and Create a Price Set as required.
  5. To the right of the Select Price Deck list, click Make Active to apply the price deck to your project.
  6. Click OK in the Price Deck Editor.
