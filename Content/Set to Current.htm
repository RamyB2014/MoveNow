Set to Current

The Set to Current button on Predictions | Declines (click ) extrapolates an existing forecast out to the Current Month + 1.

Use Set to Current in conjunction with the Properties tab (bottom left corner of the screen) to easily see if there is a difference between the Last Production Month and the Current Month. If they are different, the forecast might need to be adjusted to account for the downtime or removed if the well has been shut in or abandoned.

You can move the Properties tab to a different location on your current screen or to a separate monitor. Selecting the Current Month field on the Properties tab keeps these rows in focus while you move from entity to entity.

You can also use the Set Start of Forecast tool to move several wells (e.g. wells at a folder level) ahead at once (to the Current Month or to a user-entered date).
