Batch Apply a Type Curve to Several Wells

Use this procedure to apply a type curve to several wells at once.

You can apply the type curve on Predictions | Workspace and preview the change (or compare the change to the original decline) before actually applying the type curve to your wells. See Edit a Decline on Predictions .  

To batch apply a type curve to several wells

  1. From the Tools menu, select Apply/Remove Type Curve.
  2. Under Options, select Apply Type Curve and select Plans (to apply the changes to your project) or Workspace (to preview the changes in Workspace before applying the changes to your project).
  3. Select a Source Type Well.
  4. Select Scale forecasts when applying if you want to scale the forecast. If you don’t select this option, scaling is turned off (the scaling factor is 1).
  5. Select the Destination Plans and Reserves Categories.
  6. Click OK.

The type curve is applied to the selected wells in the selected reserves categories.

When enabled, the Apply Current Filter checkbox applies your project filter to the Destination Entities window in the Apply/Remove Type Curve dialog box. When this checkbox is disabled, your project filter is not applied to the Destination Entities window—all entities are displayed.
