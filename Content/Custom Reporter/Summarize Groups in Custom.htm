Summarize Groups in Custom Graph Reports

When a group is summarized, the graph displays a summed Y value for each point on the X axis. When a group is expanded, the graph displays all Y values for each point on the X axis.

To add a field or formula to the X axis in the group outline: From the X Axis Grouping drop-down menu, select Summarized. The graph report will have one line for each section the group created.
