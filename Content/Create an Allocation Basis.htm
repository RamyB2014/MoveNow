Create an Allocation Basis

When creating a Fiscal Regime for a Ring Fence, you might want to allocate the ring fence results to individual entities according to an allocation basis. provides a list of criteria you can select for as an allocation basis, but it also enables you to create your own allocation basis by creating basic formulas.

Once you create an allocation basis, you can select it as an allocation basis for other variables.

This procedure is performed on the Edit tab in the Fiscal Regimes Editor. To open the editor, click the Tools menu, point to Global Project Data and select Fiscal Regimes.

To create an Allocation Basis variable

  1. Click Add in the bottom left corner and select Allocation Basis.
  2. Type a name for the variable and click OK.
  3. Complete the following fields:
    • Input Style
    • Data type>
    • Unit Type
    • Calc. unit
  1. Click  in the Formula field to create the formula for the variable. See Create Formulas. See Create Formulas for Regime Variables.
  2. In the Formula Editor, create the formula by moving formula elements to the formula window.
  3. Click OK.
