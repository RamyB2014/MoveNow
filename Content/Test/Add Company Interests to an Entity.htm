Add Company Interests to an Entity

Company interests are added to wells on Economics | Interests & Royalties.

Before you can add a company interest to a well, the company must exist in the project. If the company does not already exist, see Add a Company to the Project.

To add a company interest to a well

  1. On Economics | Interests & Royalties, click Add.
  2. From the Company list, select a company.
    3. Companies in grey text can be selected but are not enabled. Companies must be enabled in the Company Editor in order to be included in economic calculations. See Add a Company to the Project.
  3. Under Lease, select a new or existing lease.
  4. Select the products for which the interest will be added and click OK.
    The new interest is displayed below the Lease in the Interests window.
  5. To the right of the Interest window, enter a working interest and other Interest Properties.
    6. You can modify which interest properties are displayed by clicking  and selecting the properties you want to display.
  6. Optional. Select the checkboxes for Start Date and/or End Date at the top of the screen and select the dates.
  7. If you do not select a Start Date, the interest will begin on the Reference Date. If you do not select an End Date, the End Date defaults to the Reference Date plus 60 years.
