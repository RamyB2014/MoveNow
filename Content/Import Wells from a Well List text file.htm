Import Wells from a Well List (text file)

Use this procedure to import wells from a PPDM data source (such as IHS or GeoLogic) based on a well list (.txt file).

To import wells based on a well list

  1. From the File menu, select Import.
  2. In the Import dialog box, select Import from data source(s) and select your data source.
  3. Under Wells to Update, click  next to Well List.
  4. Browse to the text file and click Open.
  5. Click Next and edit the Import Options, if required.
    6. Because you are importing wells based on a well list, the Wells to Update settings do not apply. Only the wells in your well list will be imported.
  6. Click Next and edit the PPDM Import Options, if required.
  7. Click Finish.
