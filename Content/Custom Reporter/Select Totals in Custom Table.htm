Select Totals in Custom Table Reports

All fields for table reports have a Report Total row in the report outline. Depending on the type of field, you can select different report total types such as count, average, totals, etc. Some types of totals are not available for some fields.

To select report totals

  1. Click a cell in any column in the Report Total row of the report outline.
  2. Click the drop-list arrow and select a report total.

The table below describes the types of reports totals.

Type of Report Total Description
Count Displays a count of how many items are in the column
Sum Displays the sum of the items in the column
Average Displays the average of the items in the column
Minimum Displays the minimum value in the column
Maximum Displays the maximum value in the column
Formula Brings up the Formula Editor where you can enter your own formula for a report total
