Apply a Type Curve to your Current Well

You can apply a type curve to your currently selected well (or you can batch apply a type curve to several wells at once). calculates an appropriate factor to match the well’s history to the type curve.

You can apply the type curve on Predictions | Workspace and preview the change (or compare the change to the original decline) before actually applying the type curve to your well. See Edit a Decline on Predictions | Workspace .  

Also see Update a Well Linked to a Type Well.

To apply a type well curve to your currently selected well

  1. Do one of the following:

  3. In the Type Well Link dialog box, click  to browse for a source type well.
  4. In the Add to Entity dialog box, select the type well and click OK.
  5. If required, edit the Scaling Factor and click . The Scaling Factor scales the type well forecast to the existing production forecast in the target entity.
  6. Click Apply or OK or Preview in Workspace.
