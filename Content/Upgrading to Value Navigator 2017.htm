Upgrading to 2018

When you open an older project (version 6 or higher) in 2018, it is converted to 2018 automatically.

SQL Server and Oracle databases upgrade "in place" which means you should have your database administrator make a backup copy of server projects before you upgrade them.
When you upgrade a SQLite database, a backup of the original project is automatically created and saved in the same folder.
