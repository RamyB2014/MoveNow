Delete Entities

When entities are deleted from a posted (reserves) database, they are not entirely removed from the database. Deleted entities are removed from all Explorer views except the Deleted Wells view (which does not display deleted Groups or Rollups) which you can select from the Entity Type list in the Standard toolbar. Only deleted wells with change records are displayed in the Deleted Entities view.

Be advised that:

  • Deleted entities cannot be undeleted.
  • The names of deleted entities can be reused, but reusing the name links the new well to the deleted entity’s reserve reconciliation records.
  • When groups and rollups are deleted, the wells inside them are not deleted.
  • If the well you are deleting is a source well for a well schedule, deleting it makes the schedule uneditable.

Delete a Single Entity

To delete an entity

  1. In the Entity Explorer, select the entity to be deleted.
  2. Right-click and select Delete Entity or select Delete Entity from the Entity menu.
  3. Click Yes to delete the entity.

Delete Several Entities

To delete several entities

  1. On the Entity menu, click Delete Entities to enter Delete Entities Mode.
    Check boxes are displayed next to the wells in the Entity Explorer.
  2. Select the wells you want to delete and click Delete at the bottom of the Entity Explorer.
To simplify the selection of several entities, filter to the entities first. See Flag and Filter Entities.
