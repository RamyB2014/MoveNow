Create or Edit Change Record Categories or Summary Types

To create, edit, delete, or deactivate a change record category

  1. On the Tools menu, point to Global Project Data and click Change Record Categories.
  2. On the Change Record Categories tab, do one of the following:
    3. To Do this, And
    Add a category, Click Add, Enter a Name and select a Summary Type.
    Edit a category, Click the category name, Edit the Name and Summary Type.
    Delete a category, Click the category name, Click Remove.
    Deactivate a category Deselect the Active checkbox next to the category. Go to step 3.
  3. Click OK.

To create, edit, or delete a change record summary type

  1. On the Tools menu, point to Global Project Data and click Change Record Categories.
  2. On the Summary Types tab, do one of the following:
    3. To Do this, And
    Add a summary type, Click Add, Enter a Name.
    Edit a summary type, Click the summary type name, Edit the Name.
    Delete a summary type, Click the summary type name, Click Remove.
  3. Click OK.
