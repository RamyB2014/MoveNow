Create a Temporary Rollup

Only one temporary rollup can exist in a project at a time. When you create a temporary rollup, any other temporary rollup is automatically deleted.

To create a temporary rollup

  1. In the Explorer, select the wells you want to create as a rollup.
  2. Right click the selected wells, point to Create as, and click Temporary Rollup.
  3. In the Create Rollup dialog box, edit the Entity Properties, if required. The Name is not editable.
  4. (Optional) To create a working interest temporary rollup, from the Factor list, select the custom field in which you entered the factor value for each entity in the rollup. See Create a Working Interest Rollup.
    5. If you select a Factor, entities without a value in the factor custom field are not included in the rollup calculations.
  5. Click OK.
