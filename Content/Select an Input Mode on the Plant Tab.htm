Select an Input Mode on the Plant Tab

On Predictions | Data | Plant, you can enter plant data in three Input Modes, depending on the type of data you want to enter.

Input Method Description
Raw ratios Use this method to enter ratios based on raw gas. These ratios are applied to the Raw gas volumes and result in Process Loss. The Yields in the gas properties grid are based on Inlet gas. When you select this input method:
  • The Inlet Gas Analysis is highlighted in blue, indicating that you can modify the analysis numbers, if required.
  • The grid below the gas analyses displays columns for Surface, Process, and Fuel Loss.
Inlet ratios Use this method to enter ratios based on plant inlet gas. These ratios are applied to the Inlet gas volumes and result in Process Loss. The Yields in the gas properties grid are based on Inlet gas. When you select this input method:
  • The Inlet Gas Analysis is highlighted in blue, indicating that you can modify the analysis numbers, if required.
  • The grid below the gas analyses displays columns for Surface, Process, and Fuel Loss.
Sales ratios Use this input method to enter ratios based on sales gas. These ratios are applied to the Outlet gas volumes and do not result in further Process Loss. When you select this input method
  • The Outlet Gas Analysis is highlighted in blue, indicating that you can modify the analysis numbers, if required.
  • The grid below the gas analyses displays columns for Surface and Fuel Loss.
