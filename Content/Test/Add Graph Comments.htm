Add Graph Comments

To add a comment to a graph

  1. On Predictions | Declines, right-click a graph and select Add Graph Comment.
  2. Select one of the following options:
    • Add Point: Adds a floating comment that can be attached to a data series or a Y value.
    • Add Vertical Line: Adds a comment that is attached to the X axis.
    • Add Horizontal Line: Adds a comment that is attached to the Y axis.
  1. Click on the graph where you want to add the comment.
  2. Type the comment text, select a colour and edit the comment location as required.
  3. If you are adding a point annotation, select one of the following:
    • Attach comment to data series: Automatically places the comment on the decline of the product below where you clicked.
    • Attach comment to Y value: Leaves the point at the same Y value as where you clicked
  1. Click OK.

The comment is added to both graphs.
