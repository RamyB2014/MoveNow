Create a Working Interest Rollup

Creating a working interest roll up requires you to create a custom field that you use to hold a Factor value. You then add a factor value to each entity in the rollup. The factor value is applied to each entity's history, forecast, daily data, P/Z, and volumetrics when the roll up is calculated.

To create a working interest roll up

  1. Create a custom field to hold the factor value. The custom field must be a Number field.
  2. On Wells | Custom Fields, enter a value in the custom field for each entity in the rollup and save the edits.
  3. Create a Rollup|topic=Create a Rollup. You can select the custom field holding the Factor value. The factor values you entered for each entity are applied to each entity and used in the roll up calculations when the rollup is created.
    4. If you select a Factor, entities without a value in the factor custom field are not included in the rollup calculations.
