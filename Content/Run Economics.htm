Run Economics

You can calculate economics from anywhere in without generating reports by clicking the Calculate Economics button in the toolbar ().

To calculate economics

  1. In the Entity Hierarchy, select the entities or folders you want to calculate.
  2. Select the Plan and Reserves Category (located below the Entity Hierarchy) you want to calculate.
  3. Click  in the toolbar.
  4. From the list of scenarios, select the Scenario you want to calculate.
  5. Click Calculate Economics.
