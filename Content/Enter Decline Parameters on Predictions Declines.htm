Enter Decline Parameters on Predictions | Declines

You can create a decline or edit the parameters of a decline on Predictions | Declines.

Use the buttons below to add or remove declines or decline segments.

Button Well Type Action
No production history, no forecast Adds a default decline.
Has production history, no forecast Calculates a forecast.
Has a forecast Adds a segment to the forecast
Any well Deletes the selected decline segment.

The symbols below are displayed on Predictions | Declines if the decline has errors or warnings. See Decline Errors and Warnings.

Button Well Type Action
Symbol Description  
Warning. Place your mouse cursor over the symbol to see the warning.  
Error. Place your mouse cursor over the symbol to see the error.  

To change the fields displayed on Predictions |Declines, see Customize Predictions .
