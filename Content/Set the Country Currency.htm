Set the Country Currency

The country currency is the currency assigned to a country. has two countries and three currencies created by default: Canada (with Canadian dollars), the US (with US dollars) and the Euro. You can add more countries and currencies. You can also change the default currencies assigned to each country.

If you need to add countries or currencies, see Add a Country or Add a Currency.

To set the country currency

  1. On the Tools menu, select Global Project Data and click Countries & Currencies.
  2. In the Countries & Currencies Editor, click the Countries tab.
  3. In the country list, click a country.
  4. From the Currency list, select the currency you want to assign to the country.
  5. Click OK.
 When you create a new entity and assign it a country, the entity inherits that country’s currency by default. Therefore, if you change a country`s currency, all entities in that country with currency set to Default (currency) will inherit the new currency. The currency of entities in that country will not change if the entity currency is not set to Default. See Set the Entity Currency.
