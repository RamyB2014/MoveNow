Create a Stream Price

The Sample price deck included with includes bench mark stream prices that you can use as a basis for creating your own stream prices. Once you have created stream prices for all of your products, you can group the stream prices into price sets and apply the set to an entity or a set of entities. See Create a Price Set.

To create a stream price

  1. In the Tools menu, select Global Project Data > Price Decks.
  2. In the Price Deck Editor, click Stream Prices (on the left).
  3. In the editor window, select the folder level where you want the new stream price.
  4. Under the Stream Prices window, click Add.
  5. Enter a name and click OK.
  6. To the right of the Stream Prices window, specify the Country, Province/State, (these fields are already selected if you chose the appropriate folder level) Product, and Currency.
  7. Do one of the following:
    8. Options Go to
    Use a base price as a starting point, and adjust it with differentials. Step 8.
    Enter prices in the date array, starting at the Reference Date. Step 14.
  8. Under Parent Price, click Set and select a price.
    9. If the Stream Price Currency does not match the Parent Price currency, an exchange column is automatically inserted between the Parent and Result price and the exchange rate is applied.
  9. Under Differentials, click Add.
  10. Enter a name and Description.
  11. Select a Differential Type and click OK.
  12. Enter the differential value in the new column between the Parent price and the Result price.
  13. Optional: To add a price escalation, complete the steps below. See Escalate Prices for more information.
    • In the Escalation % row in the header area of the Result column, enter an Escalation %.
    • In the Result column, enter a new price in the year the escalation begins.
  1. The cell is displayed with a grey background to indicate a manual price override. The Escalation % is applied to the override and begins one year after the override.
  2. Click Save.
