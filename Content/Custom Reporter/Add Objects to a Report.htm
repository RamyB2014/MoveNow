Add Objects to a Report

A report object is a table or graph that appears in a report. A report can have more than one report object. Once you have created a report with a report object, you can add additional report objects.

To add objects to a report

  1. In the top left of the Custom Report Editor, click to add a table object or click to add a graph object.
  2. In the New Report Object dialog box, select the type of information you want to add to the report and click OK.
