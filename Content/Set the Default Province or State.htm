Set the Default Province or State

When you set a province or state as the default, the Province/State field is automatically populated when you create a well. If you import a well from another project, the well’s Province/State is retained.

To set the default province/state

  1. On the Tools menu, click Options.
  2. In the Options dialog box, click the Project Options tab.
  3. On the left, select General.
  4. On the right, select the province or state from the Default state/province list and click OK.
