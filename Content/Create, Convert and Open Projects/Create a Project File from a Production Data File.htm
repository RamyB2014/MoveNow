Create a Project File from a Production Data File

Project Files are single-user, file-based projects that you can store on your computer and email to other users.

To create a project file from a production data file

  1. On the File menu, click Open.
  2. In the Project Launcher, click Create from Import.
  3. Browse to the production data file and click Open.
  4. In Project Name, accept the default name or type a new one.
  5. In File Path, accept the default location or browse to a different location to save the new project file.
  6. From the Type list, select Project File.
  7. Click OK.
  8. In the Import Options dialog box, click Edit to change the options or click Next to continue.
    9. Depending on the last production dates in your data file, you might be asked to set the current month.
  9. Click Finish.

A new project is created in the location you specified and opens. The new project appears in the Project Launcher the next time you open it.
