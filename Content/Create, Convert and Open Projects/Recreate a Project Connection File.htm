Recreate a Project Connection File

If you accidentally delete a project .vndl file (for Oracle and SQL Server projects), you can re-establish the link by recreating the project connection file.

You can also remove the password from a connection file if you want to share it but require other users to enter a password. See Remove the Password from a Project Connection File.

To recreate a .vndl file

  1. Launch and close the Project Launcher. If you have a project open, click  to close the project ( remains open).
  2. From the File menu, select Edit or Create a Project Connection File.
  3. In the Project Connection Information dialog box, select the Project Type.
  4. Do one of the following:
    5. To Do this
    Recreate an Oracle connection, Complete the Server, User (Oracle schema), and (Oracle schema) Password fields.
    To recreate a SQL Server connection, Complete the Server and Project fields.
  5. Click Test Connection.
  6. If the connection is successful, click Save and Close.
  7. Click .
  8. In the Project Launcher, click Browse.
  9. Browse to the .vndl file you created and click Open.
