Create Custom Capital or Operating Costs

In addition to the default operating and capital costs in , you can also create custom costs.

To create a custom cost

  1. In the Tools menu, point to Global Project Data and click Costs.
  2. In the Costs dialog box, click Add.
  3. Complete the following fields:
    4. Field Description
    Name Enter a name for the cost.
    Country Specify a country for the cost. This determines the cost Categories, below.
    Category Select a cost category.
    Interest Type Select Working Interest or Facility.
    Success tax pool Select a success tax pool.
    Failure tax pool Select a failure tax pool.
    Include in reversion calculations Costs are included in reversion calculation.
    Visible when selecting new entity costs The cost is only visible for selection when this option is selected. If the cost has been added to entities, it is still visible even if this option is not selected.
  4. Click Save.
