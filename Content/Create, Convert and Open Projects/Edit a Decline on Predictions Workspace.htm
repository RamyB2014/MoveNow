Edit a Decline on Predictions | Workspace

To edit a decline in Predictions | Workspace

  1. On Predictions | Workspace, click the forecast you want to edit and click Edit.
  2. In the Edit Forecast dialog box, edit the decline parameters or apply a type curve.
  3. Do one of the following, if required:
    4. To... Do this...
    View the edited decline in the Workspace, Click Preview in Workspace.
    View the original version of the edited decline in the Workspace, Click Preview Original in Workspace.
  4. Click OK.
