Apply a Forecast to a Reserves Category and Plan

After you create a forecast on Predictions | Workspace, you can apply it to several plans and reserves categories on the current entity.

When you apply a forecast to a plan and reserves category, a copy of the forecast is made and applied to the new plan. The copied forecast’s name is the same as the original, but the Plan and Reserves Category columns are populated and the text is bold. This is the active forecast for the reserves category and plan and is also displayed on Predictions | Declines when you select the corresponding reserves category and plan. The original forecast is still displayed on Predictions | Workspace, but its text is not bold and the Plan and Reserves Category columns are not populated. You can apply the original forecast to other plans and reserves categories, which always creates a copy as described above.

To apply a workspace forecast to more than one plan or reserves category for several wells at once, see Apply Workspace Forecasts.

To apply a forecast to a plan and reserves category

  1. Select the forecast you want to apply to a plan and reserves category.
  2. Click Apply Forecast.
  3. Select the following options:
    4. Option Description
    Plan The plan in which the forecast will be active. See Create a Plan.
    Reserves Category The reserves category in which the forecast will be active.
    Create a copy of the target active forecast Only available if you have selected a plan and reserves category with an active forecast. This option creates a copy of the currently active forecast, which you are replacing with the new forecast. The copy takes the name of the name of the forecast being replaced and appends “Copy” to the name. If the forecast being replaced has no name, the copy is named “<Plan>+<Reserves Category>+Copy”. If you do not select this option, the forecast being replaced is overwritten by the new forecast—it is not saved.
  4. Click OK. The active forecast is displayed in bold text.
