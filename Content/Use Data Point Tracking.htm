Use Data Point Tracking

Data point tracking is used to display the exact values on graphs at the intersection of the X and Y axis. Data point tracking can be used on Predictions | Declines, Workspace, P/Z, or Volumetrics. When the data point tracker is turned on, it is included in printed graphs.

To use data point tracking

  1. Do one of the following:
    • Press Ctrl + T.
    • Right-click a graph and select Data Point Tracking.
    • Click  on the Graph toolbar.
  1. In the Data Point Tracking dialog box that is automatically displayed, select the data you want displayed.
  2. Do one of the following to find the value of a data point:
    • Move the mouse pointer over the point until the data is displayed.
    • Move the data point tracker by pressing the Left and Right Arrows on the keyboard.
Use the Up and Down arrows to lock and unlock the data point being displayed.
