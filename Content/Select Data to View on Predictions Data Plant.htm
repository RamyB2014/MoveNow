Select Data to View on Predictions | Data | Plant

Use the table below to edit the grid view or view data.

To view Do this 
A specific product, Select an item from the Products list.
Only one of the following on the grid: efficiencies, volumes, ratios, rates, or input data, Select an item from the View list.
History only, Forecast only, or both, Select an item from the list to the right of the View list.
Monthly data or only sparse data (Sparse displays data only in months with a change in data), Select Monthly or Sparse from the list under Display mode in the bottom right of the screen.
The end of Actual data, Click End of Actuals in the bottom left of the screen.
The Forecast Start, Click Forecast Start in the bottom left of the screen.
The Calculation Date, Click Calculation Date in the bottom left of the screen.
