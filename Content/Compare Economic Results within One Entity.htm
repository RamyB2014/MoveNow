Compare Economic Results within One Entity

Use this procedure to compare results between archives, plans, reserves categories, or scenarios within an entity.

To compare results within one entity

  1. From the Comparison list at the top of Comparison | Economic, select one of the following: Cash Flow, Costs, or Production.
  2. Select a Currency, if required.
  3. Select the required Display Options.
  4. In the Explorer, select an entity or folder, a reserves plan, plan, or calculated plan, and a reserves category.
    The entity or folder is displayed in bold in the list at the bottom of the Comparison tab, prefixed with “Current”.
  5. Click Add Row.
    There are now two “Current” entities in the comparison list.
  6. Repeat step 5 to add more current entities.
  7. In the second current entity, change the archive, plan, reserves category, or scenario and repeat this step for any other entities in the comparison list. The results of the comparison are automatically displayed.
    8. If you do not pin the original “current entity” to the list (right-click the entity and select Pin selection) it will not be retained when you edit the archive, plan, reserves category or scenario of the added rows).
  8. Optional. Click Quick Add to add new rows with the same entity, but with different Reserves Categories or Plans, and select one of the following two options:
    9. Option Description
    All reserves categories in <Plan> Adds all reserves categories with data for the current entity in the current Plan.
    <Reserves category> in all plans Adds the current reserves category for the current entity in all Plans to the list. 
  9. Optional. Set a row as the baseline variance on the Comparison Tab.
  10. Optional. Click the Play button in the main toolbar to play through all wells. The comparison results are displayed for each entity.>
