Find and Update Wells Linked to a Type Well

When you have edited a type well’s forecast, you must update any wells linked to the type well because the new type well forecast is not automatically applied to any linked wells. Use the procedure below to display all the wells linked to a type well and update their forecasts.

To display and update all wells linked to a type well

  1. From the Entity Type list in the main toolbar, select Type Wells.
  2. In the Entity Explorer, select the appropriate type well and go to Type Well | Links.
    All wells linked to the current type well are displayed.
  3. If required, filter the list to the entities, plans, and reserves categories you want to update by clicking above the column headings and applying column filters.

    4. Note: Do not use flags to select the wells you want to update—only use the column filters.

  4. Click Apply Type Curve.

The updated type well forecast is applied to all the wells in the list in the selected Plans and Reserves Categories.
