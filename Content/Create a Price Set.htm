Create a Price Set

A price set is a collection of stream prices for several products. Before creating a price set, you must create each of the stream prices you want to include in it. See Create a Stream Price.

To create a price set

  1. In the Tools menu, select Global Project Data >Price Decks.
  2. In the Price Deck Editor, click Price Sets (on the left).
  3. In the editor window, select the folder level where you want the new price set.
  4. Under the Price Set window, click Add.
  5. Enter a name and click OK.
  6. To the right of the Price Set window, specify the Country and Province/State, (these fields are already selected if you chose the appropriate folder level).
  7. Under Prices, select a stream price for each product.
  8. Click Save.
