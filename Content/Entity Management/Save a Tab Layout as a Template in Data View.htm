Save a Tab Layout as a Template in Data View

Tab layout determines how the data in the grid is displayed. Layouts determine which fields are displayed in the rows and columns of the grid, which data are shown in the grid (as [Values]), and which filters are applied to the data in the grid and within each field.

Once you have created and configured a tab in Data View, you can save the layout as a template and reuse it or share it with other users.

To save a tab layout as a template in the Data View

  1. In the top right corner of the layout panel, click Save As Template.
  2. Under Location in the Save As Template dialog box, select a folder.
  3. If required, click New Folder to create a new folder in the location you selected.
  4. If required, type a new Name.
  5. Click Save.
