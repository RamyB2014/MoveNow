Normalize a Type Well Curve to Peak Rates

To normalize the type well curve to the peak production of each child well,

  1. From the Entity Type list in the main toolbar, select Type Wells.
  2. Select the type well to edit.
  3. On the Type Well tab, click  (Normalize to Peak Rate).
  4. Click Recalculate to obtain the new type curve.

Edit the Normalization for One Well

If you want to normalize a well’s history to a different data point in its history, use the Normalization Mode tool to drag a well’s curve on the Type Well graph.

To normalize a well

  1. From the Entity Type list in the main toolbar, select Type Wells.
  2. Select the type well to edit.
  3. On the Type Well tab, click  (Normalization Mode).
  4. Select a well to edit by placing your mouse pointer near a selectable point (indicated by a dot on the curve where the decline rate changes).
  5. When the pointer changes to a hand, hold the left mouse button and drag the curve until the point you want to use is over the dotted line.
  6. Click Recalculate.

Normalize Range

To select a range of data on which to base the type curve, use the Normalize Range tool on the Type Well tab.

To select a range of data to normalize

  1. From the Entity Type list in the main toolbar, select Type Wells.
  2. Select the type well to edit.
  3. On the Type Well tab, click  (Normalize Range).
  4. Place your pointer above the start or finish point of the period you want to use.
  5. Hold the left mouse button and drag the pointer horizontally across the range you want to use.
  6. Release the mouse button and click Recalculate.

Set Normalization Back to Zero

To set the normalization of all wells in a type well back to zero (the original type well calculation), click  (Reset to Zero) on the Type Well tab.
