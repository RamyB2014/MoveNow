Enter Modernized Royalty Framework (MRF) Parameters

To enter MRF parameters go to Economics | Interests & Royalties. You can also enter some MRF parameters in the Data View on Interests & CDN Royalties | Alberta Royalties.

To use the Emerging Resources Program incentive type, create a Ring Fence for the eligible wells and select the Emerging Resources Program from the Calculation option on the Ring Fence definition.

You must click the Refresh button after editing any MRF values.

To enter MRF parameters

  1. For Production Category, select one of the modernized options.
  2. For Incentive Type, select Modernized.
    The MRF parameters are displayed.
  3. Complete the following fields as required.
    4. Field Description
    Proppant Equivalency Factor This field is populated after entering the Completion Type and Acid Concentration (if applicable). 
    C* If you know the C* amount, enter it here. Otherwise, this amount is calculated once you have entered the other parameters.
    Balance Used If you know the Balance used amount, enter it here. Otherwise, this amount is calculated once you have entered the other parameters.
    Remaining Balance If you know the Remaining Balance amount, enter it here. Otherwise, this amount is calculated once you have entered the other parameters.
    Balance Effective Date Enter the date the balance was calculated.
    Incremental C* Use the Incremental C* Plug-in to calculate this value and enter it here (Plug-ins menu). Any ARF Incentives in effect when a re-entry is performed will run concurrently with the Incremental C* Incentive.
    Re-entry Date Enter the re-entry date
    Measured Depth For guidance on the values to enter for these fields, see the Alberta government website at http://www.energy.alberta.ca/About_Us/Royalty.asp
    Total Measured Depth
    True Vertical Depth
    True Vertical Depth Avg
    True Vertical Depth Max
    Lateral Length
    Total Lateral Length
    Completion Type Select a proppant.
    Total Proppant Placed Enter the total in long tons or barrels (acid only).
    Acid Concentration Complete only if you selected Acid for Completion Type.
    Oil Royalty Type Select a type.
  4. Click the Refresh button next to Incentive Type.
