Add Entities to a Group

To add a well to a group

  1. In the Explorer, select the entity you want to add to a group.
  2. Right-click the entity, point to Add to, and click Group.
  3. In the Add to Entity dialog box, select the group and click OK.

To view groups in the Entity Explorer, select Wells and Groups from the Entity Type list in the Standard toolbar.

If an entity you are adding to the group is already in another group or common termination entity, a message will ask you if you want to remove it in order to move it into the group. The well cannot exist in more than one group or common termination entity.
