Clear Workspace Forecasts

You can clear Predictions | Workspace forecasts by selecting Clear Workspace Forecasts from the Tools menu. Administrators can clear forecasts for all users. Normal users can only clear their own forecasts (if they have the Manage Database Tables security policy).

Cleared forecasts are permanently deleted.

To clear forecasts from Predictions | Workspace

  1. On the Tools menu, select Clear Workspace Forecasts.
  2. Select a date. All forecasts on Predictions | Workspace with a last modified date prior to this date are displayed. 
  3. Select Sort by user or Sort by forecast name.
  4. If you want forecasts for child wells to be deleted, select Apply to Children of Groups.
  5. Click OK.
