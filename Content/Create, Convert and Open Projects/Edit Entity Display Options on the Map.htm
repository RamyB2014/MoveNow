Edit Entity Display Options on the Map

You can customize which entities are displayed on the map and how they are displayed. For example, you can reduce the number of entities displayed for a less cluttered view by enabling the option to only display entities in the currently selected folder.

To edit how entities are displayed on the map

  1. On the Map tab, click  and then click .
  2. Under Entity Layers, select one of the following options:
    3. Option Description
    Well Spots Entities are displayed as dots. Currently selected entities are displayed as black dots. Other entities are displayed as white dots.
    Markers Entities are displayed with a marker symbol. Marker symbols are more useful when a map contains a large number of overlapping entities. Markers indicate when there are several entities in a small area and enable you to scroll though the entities by placing your cursor over the marker and pressing the space bar or the Page Up and Page Down buttons.
    Horizontal deviation Horizontal wells are displayed on the map with a square to indicate the surface location with a line leading to the bottomhole location, which is displayed with the corresponding well symbol. Also see View Horizontal Deviation on the Map.
    To edit which entities are displayed on the map
  3. On the Map tab, click  and then click .
  4. Under Entity Options, select the following options:
    5. Option Description
    All entities in Entity Explorer Displays all entities visible in the current map view.
    Apply current filter If enabled, applies the currently applied filter to the visible entities.
    Selected folder Displays all entities in the currently selected folder. Does not display the children of parent entities unless you also select Children, below.
    Selected entities Only displays currently selected entities.
    Children Displays parent and child entities when All entities in Entity Explorer or Selected folder are selected.
    Relationships Displays a black line between the parent and child entities when the parent is selected. Displays a red line between the child and parent entities when the child is selected. Also see View Relationships for Grouped Entities on the Map.
  5. Click OK.
