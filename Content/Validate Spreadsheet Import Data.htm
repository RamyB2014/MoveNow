Validate Spreadsheet Import Data

Once you are ready to import your data, you should validate it by clicking Validate in the bottom right. Validation helps ensure that you have correctly configured the Spreadsheet Import column headers and that the data can be imported (the wells exist in the project, for example).

When you click Validate, a log file is created. If you receive a validation error or warning, click Open log next to the Validate button to view the contents.

The validation log is only available during the current Spreadsheet Import session. When you close the Spreadsheet Import tool, the log file is not saved. If you need the log file for future reference, click File > Save or Save As.
