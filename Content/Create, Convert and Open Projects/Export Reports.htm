Export Reports

To export a report on the Reports tab or in the Print Preview (accessed from the Batch Manager)

  1. Click Export .
  2. In the Export Report dialog box, type a name for the file.
  3. From the Save as type list, select the format you want to save the file in (.pdf, .csv, or .html).
  4. Click Save.

To export all reports in the Print Preview (accessed from the batch Manager)

  1. Click Export All .
  2. In the Export Report dialog box, type a name for the file. When you choose to export the reports as individual files, this name is used to create a folder that contains the reports.
  3. Do one of the following:
    4. To Do this
    Save each report as a file, From the Save as type list, select the format you want to save the files in (.pdf, .csv, or .html).
    Save all reports as one file, From the Save as type list, select PDF File (single file).
  4. Click Save.
