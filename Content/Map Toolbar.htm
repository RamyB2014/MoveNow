Map Toolbar

The map toolbar is displayed on the Map tab.

Icon Icon Name Description
Manage Layers Opens the Layers configuration panel.
Pan Mode* Click to grab the map and move to it to view the desired area.
Selection Mode* Click, hold and drag to select a portion of the map.
Zoom mode* Click, hold and drag to select an area of the map to zoom in to.
Zoom to project Zoom and re-center map to include the entire current project.
Zoom to selection Zooms to the entity selected either in the Entity Hierarchy, or on the map.
Zoom In Click to zoom in on the map by increments of two.
Zoom Out Click to zoom out from the map by increments of two.
Go to location Click to open the Pan to Location dialog box. See Pan to Location.
Show legend Displays or hides the map legend when Data Visualization is enabled. .
Print Opens the Print preview
Refresh Refreshes the entire map and all layers.
Pop out map Opens the map in a new window.
Go offline Click to go offline. When you go offline, use this button to go back online. When you go offline, ValNav does not automatically connect when restarted or when a connection becomes available.
Stop currently running map operations Cancels map operations.
About the map Displays a dialog with basic map parameters and diagnostic information

The Select, Zoom, and Pan functions are also available through shortcut keys. See Map Shortcut Keys.
