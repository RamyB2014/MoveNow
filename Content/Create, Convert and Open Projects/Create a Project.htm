Create a Project

You can create an empty project or you can use a production data file (such as .axp) to create a project with production data.

There are four project types:

  • Project File: A single-user, file-based project you store on your computer and can email to other users.
  • Oracle: You need Oracle administrator rights or assistance from a DBA to create Oracle projects. Contact your administrator.
  • SQL Server: You need SQL Server administrator rights to create an SQL Server project. When accessing the database from outside , you will need a password.
  • SQL Server - Windows Authentication: You need SQL Server administrator rights to create an SQL Server project. When accessing the database from outside , Windows Authentication is used.

Before creating an Oracle or SQL Server project, you need to configure the connection settings in User Options: Connection Settings.
