Add a Default Decline to a Well

For new wells without production history, you can quickly add a default decline and then edit the parameters on Predictions | Declines.

To add a default decline

  1. On Predictions | Declines, click the plus sign next to the decline type you want to add.
  2. Edit the Start Date and other parameters as required.
  3. Click the decline name (such as Oil or Gas) to display the decline Calculator where you can add more segments.
  4. Click  to edit or apply the following decline options:
    • Reserve Life Index
    • On-Time
    • Type Well Link
    • Set Start of Forecast
    • Best fit or refit
    • Recalculate a Group Forecast
    • Set to Current
    • Apply Volume
    • Unit Options

To change the fields displayed on Predictions |Declines, see Customize Predictions.
