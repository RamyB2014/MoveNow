Create a Well on the Map

Map entities can be created either directly on the map, or in the Entity Hierarchy:

To create a map entity

  1. Right-click in the map where you want the new entity created and select a well type to open the Well Information dialog box.
    The latitude and longitude in the Well Information dialog box are populated based on the map location you clicked to create the new well.
  2. Enter the necessary Well Identification information and assign the Attributes.
  3. Click Save New.
    The new well is displayed on the map and in the Entity Hierarchy.
