Open and Edit Spreadsheet Import Spreadsheets

In the Spreadsheet Import tool, you can open .xlsx, .xlsm, or .csv files. Once you have opened the file, you can also select the Sheet you want to import.

The data displayed in the Spreadsheet Import tool is only a preview.

If you need to edit the data, you must open the spreadsheet, edit it, click Save, and then click  in the Spreadsheet Import tool to refresh the preview. You can open the spreadsheet from the Spreadsheet Import tool by clicking  next to the Sheet list.

Your spreadsheet will contain column headings that are not part of the data you want to import. To exclude those rows, in the Spreadsheet Import tool, enter the number of rows to be excluded in Header rows (below the preview).
