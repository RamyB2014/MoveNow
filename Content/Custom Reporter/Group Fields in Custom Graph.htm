Group Fields in Custom Graph Reports

Grouping fields in a report makes the report easier to read by organizing the data into groups. For each group that is added, a separate line is created. 

To group fields in a custom graph report: Drag fields and formulas from the Fields and Formulas window into the Group Fields window.

Group Fields can also be summarized or nested. See Summarize Grouped Fields in Custom Table Reports and Nest Group Fields in Custom Graph Reports.
