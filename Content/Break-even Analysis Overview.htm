Break-even Analysis Overview

The Break-even Analysis (BEA) tool is used to determine the price at which a set of entities has an NPV of zero—the break-even price.

The break-even price is an economic indicator that can be used for:

  • Understanding the price sensitivity of a project
  • Ranking projects for budgeting purposes
  • Planning your portfolio

The BEA runs multiple iterations starting with the base price and runs until it reaches a price that returns an NPV of zero (or until further iterations no longer result in a value closer to zero). The results of the iterations are displayed for all products, with the primary product supply cost shown at the top.
