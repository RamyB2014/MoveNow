Perform a Volumetric Analysis

To calculate reserves using volumetrics

  1. Go to Predictions | Volumetrics.
  2. Enter all required fields (yellow).
  3. If required, calculate the water saturation and Z factor. See Calculate Water Saturation and Calculate Z Factor.
When you enter a value that has other values linked to it, some fields might turn red. Fields with red text need to be recalculated. To recalculate a red field, select it and press F9.

To link a decline to a volumetric analysis, see Link a Decline to a P/Z or Volumetric Analysis. It is possible to simply enter OGIP or OOIP and a recovery factor, if more granular parameters are not available.
