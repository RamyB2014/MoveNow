Delete a Price Set

To delete a price set

  1. In the Tools menu, point to Global Project Data and click Price Decks.
  2. In the Price Deck Editor, click Price Sets.
  3. Select the price set you want to delete.
  4. Below the price deck editor window, click Remove.
  5. In the Delete Price Set dialog box, click Yes.
Deleting a price set removes it from all price decks.
