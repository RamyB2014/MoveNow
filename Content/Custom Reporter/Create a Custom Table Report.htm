Create a Custom Table Report: Overview

To create a new table report

  1. From the Tools menu, select Custom Reports or click on the Standard toolbar.
  2. Click Add to open the New Report Object dialog box.
  3. Select Table.
  4. Select the Type of information to report and click OK.
  5. In the Report Editor, Add Fields and Formulas to Custom Table Report Objectsto the report.
  6. Group Fields in Custom Table Reports.
  7. Format Custom Table Report Properties.
  8. Add Filters to Custom Table Reports.
