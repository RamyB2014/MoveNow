Edit On-Time

You can select one of the on-time types below by clicking  on Predictions | Declines:

  • Constant: Standard on-time. Used for all time-steps. This type of on-time is updated when production is imported if Recalculate Constant On-time on Import is selected in User Options: Import Parameters. Constant on-time is calculated using the most recent 6 months of production history.
  • Recurring: Used to specify the on-time for each month. The monthly values are used on a recurring basis for the life of the well.
  • Manual Dates: Used to specify on-time for specific periods of time using a sparse entry grid. The first value entered is used until a new value is entered.
