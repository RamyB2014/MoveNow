Create a Cost Entity

To create a cost entity

  1. In the Entity Explorer, right click, point to Create and select Oil Well or Gas Well (it doesn’t matter which you choose).
  2. In the Well Information dialog box, complete the fields below.
    3. The fields below are the minimum fields required to create a cost entity. Complete additional fields as required.
    Tab Field Description
    Well Properties Display Name Name displayed in the Entity Explorer.
    Field Field
    Pool Pool
    Country Select the country. See Add a Country.
    State/Province Select the Province or State. See Add a Province or State.
    Well Type Select Cost Entity.
    Entity Currency Select the currency for economic calculations. See Add a Currency.
  3. Select the cost entity in the Entity Explorer and go to Economics | Prices & Costs | Capital Costs to add capital costs. See Enter Capital Costs.
  4. Go to Economics | Prices & Costs | Operating Costs to add operating costs. See Enter Operating Costs. Only add fixed operating costs to a cost entity.
    5. Cost entity costs inherited from another reserves category or plan are not included in economic calculations. Only costs entered directly in the reserves category and plan are included.
  5. For all operating costs, change the Cost Type to Cost/Time.
