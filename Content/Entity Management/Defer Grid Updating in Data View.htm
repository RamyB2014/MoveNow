Defer Grid Updating in Data View

When you are changing the layout of a tab in the layout panel, your changes are automatically applied in the grid. If you want to prevent changes from being automatically applied as you work, you can defer the changes by selecting Defer grid update at the bottom of the layout panel. Your layout changes will not be applied until you click Update.

To defer grid updates in the Data View

  1. Select the appropriate tab and click  to display the layout panel.
  2. At the bottom of the layout panel, select Defer grid update.
  3. When you want to update the grid data, click Update.
