Add Comments to an Entity

To add comments and/or link documents to an entity

  1. In the Entity Explorer, select the entity to which you want to add a comment.
  2. Go to Info | Comments and click Add.
  3. Select a Comment Type and Reserves Category.
  4. Type a comment.
  5. To link a document to the entity, click  next to Link.
  6. Linking a document creates a hyperlink to the document’s location. It does not attach the document.
  7. Browse to the file and click Open.
  8. Click OK.
