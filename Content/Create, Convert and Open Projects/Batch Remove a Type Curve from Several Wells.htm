Batch Remove a Type Curve from Several Wells

Use this procedure to remove a type curve from several wells at once.

You can remove the type curve on Predictions | Workspace and preview the change (or compare the change to the original decline) before actually removing the type curve from your wells. See Edit a Decline on Predictions .  

To batch remove a type curve from several wells

  1. From the Tools menu, select Apply/Remove Type Curve.
  2. Under Options, select Remove Type Well Link.
  3. Select the Destination Entities and Destination Reserves Categories.
  4. Click OK.

The type well link is removed from the selected wells in the selected reserves categories.

When enabled, the Apply Current Filter checkbox applies your project filter to the Destination Entities window in the Apply/Remove Type Curve dialog box. When this checkbox is disabled, your project filter is not applied to the Destination Entities window—all entities are displayed.
