Create a Decline on Predictions | Workspace

When you create a decline Predictions | Workspace, it is not automatically applied to a plan or reserves category and only resides on Predictions | Workspace.

When you apply a forecast to a plan and reserves category, a copy of the forecast is made and applied to the new plan. The copied forecast’s name is the same as the original, but the Plan and Reserves Category columns are populated and the text is bold. This is the active forecast for the reserves category and plan and is also displayed on Predictions | Declines when you select the corresponding reserves category and plan. The original forecast is still displayed on Predictions | Workspace, but its text is not bold and the Plan and Reserves Category columns are not populated. You can apply the original forecast to other plans and reserves categories, which always creates a copy as described above.

To apply a decline to a plan and reserves category, see Apply a Forecast to a Reserves Category and Plan.  

To create a decline

  1. Go to Predictions | Workspace and click Add Forecast.
  2. Complete the following:
    3. Field Description
    Name Forecast name
    Reserves Category Select the Reserves Category.
    Based On Only available if you have selected a Reserves Category. Links the forecast to a P/Z analysis or Volumetric analysis. See Link a Decline to a P/Z or Volumetric Analysis.
    Type Well Link Links to a Type Well’s forecast.
    Description Add a description, if required.
    On-time You can select different on-time types. See Edit On-time.
    1+WOR Water Oil Ratio.
    GOR Gas Oil ratio.
    RLI Reserve Life Index.
  3. Click the decline name (such as Oil or Gas) to display the decline calculator. See Calculate Decline Parameters.
  4. Click Preview in Workspace to view the new decline in the Workspace.
  5. Click OK.

The decline is displayed on Predictions | Workspace.
