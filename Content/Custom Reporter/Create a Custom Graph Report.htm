Create a Custom Graph Report: Overview

To create a new graph report

  1. From the Tools menu, select Custom Reports or click .
  2. Click Add to open the New Report Object dialog box.
  3. Select Graph.
  4. Select the Type of information to report and click OK.
  5. In the Report Editor, add fields (columns) to the report.
  6. Group the fields.
  7. Format the report.
  8. Add filters to the report.
