Add a Country

comes with two countries already created: US and Canada. Use the following procedures to add more countries, province, and states.

To add a country

  1. On the Tools menu, select Global Project Data and click Countries & Currencies.
  2. In the Countries & Currencies Editor, click the Countries tab.
  3. Under the country list, click Add.
  4. In the Country Name dialog box, type a name and click OK.

To add provinces or states to a country

  1. Under the States/Provinces list, click Add.
  2. In the States/Provinces Name dialog box, type a name and click OK.

To assign a currency to a country, see Set the Country Currency.
