Filter Declines on Predictions | Workspace

To filter columns on Predictions | Workspace

  1. Place your mouse in the grey area above the column header and then click the down arrow that is displayed.
  2. Do one of the following:
    3. To Do this
    Filter to any of the values displayed in the filter list, Select or deselect the checkboxes beside the values and go to step 5.
    Use the text filter, Click Text Filter below the values and go to step 3.
  3. Under Show rows, select an operator and then enter the value on the right.
  4. If required, add another operator (using and/or) and then enter the value on the right.
  5. Click Apply.
