Copy Reserves Plan Data

When you copy entity data from one reserves plan (working, pending, and accepted) to another, the entity data must be balanced. See Balance Entities.

To copy reserves plan data

  1. In the Entity Explorer, select the entity you want to copy.
  2. In the Summary window, select the reserves plan containing the data you want to copy.
  3. From the Change Management menu, select one of the following: Copy to Working, Copy to Pending, or Copy to Accepted.
