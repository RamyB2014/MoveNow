Create and Manage Forecast Names

You can create names to apply to declines you create on Predictions | Workspace. These names can help you categorize your declines. The name you apply to a forecast is specific to that entity. Therefore, you can use the same name (such as “Best Forecast”) for more than one well. Once you have created and labelled forecasts for several wells, you can also apply the forecasts with that name to different plans and reserves categories. See Apply Workspace Forecasts.

To create or edit names

  1. On Predictions | Workspace, click Manage Forecast Names.
  2. Do one of the following:
    3. To Do this
    Add a new name, Click Add and enter a name.
    Edit a name, Double-click the name and edit it.
    Delete a name, Click the name and click Remove.
    Deactivate a name (but not delete it), Deselect the Active checkbox next to the name.
  3. Click OK.
