Create a Group

To create a group

  1. In the Entity Explorer, select the wells you want to group.
  2. Right-click the selected wells and click Create as Group.
  3. Type a name for the group
  4. Under Phase, select Oil or Gas.
  5. Under Calculate Reserves Categories, select the reserves categories you want to calculate.
  6. Edit other group information, as required, and click OK.

Custom fields are populated based on the wells that belong to the group, but this can be edited if necessary.
