Manage Entity Data

You can use the Data Manager to copy reserves category data to other reserves categories, merge plans, or remove inputs.

To access the Data Manager, do one of the following:

  1. Right-click in the Summary window and select Copy Reserves Category Data, Advanced Copy, Merge Plans, or Remove Inputs.
  2. Select Data Manager from the Entity menu.
  3. See the topics below for more information on each tab in the Data Manager.
