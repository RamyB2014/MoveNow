Create a Calculated Plan

To create a calculated plan

  1. On the Tools menu, point to Global Project Data and click Plans.
  2. In the Plans dialog box, click the Calculated Plans tab.
  3. Click Add, and select the Plans you want to include in the Calculated Plan (you can also add or remove plans later).
  4. Click OK. The new Calculated Plan is displayed with the default name New Calculated Plan.
  5. Edit the name and add a Description, if required.
  6. Under Components, specify the wedge calculation between the plans by editing the Factor (a plan with a factor of -1 will be subtracted from a plan with a factor of 1, for example).
  7. Add or remove Plans, if required, by clicking Add Plans or Remove.
  8. Click OK.
