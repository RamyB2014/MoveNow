Delete Reserves Category Data

There are two methods for deleting forecast and economic data specific to a reserves category. Method 1 is used to delete data from one reserves category in one entity. The second method is used to delete data in several reserves categories for one or several entities.

Method 1

To delete all forecast and economic data for a single entity in a single reserves category

  1. In the Summary window, click  in the Reserves Category for which you want to delete all data.
  2. In the warning message, click OK to confirm the deletion.
  3. Volumetric data is not deleted. It must be deleted manually from the Predictions | Volumetrics tab.

Method 2

If you want to delete data for several entities, do one of the following first:

  • Select the entities in the Entity Explorer (using the Shift key to select consecutive entities or the Ctrl key to select non-consecutive entities)
  • Select the folder level of the entities you want to copy
  • Filter to the entities

Go to Remove Inputs.
