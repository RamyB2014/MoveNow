Reserves Variance Report

The Reserves Variance report summarizes reserves and NPV (at two discount rates) for two selected reserves plans.

To run the Status Variance report

  1. From the Summary window, select the reserves plan you want to compare.
  2. From the Change Management menu, select Reserves Variance.
  3. Select a scenario (or jurisdiction).
  4. Select Run Economics, if required.
  5. Select Preview on Selection (so that the selected entity or folder is previewed automatically when you select it when in the Print Preview).
  6. In the To list, select the reserves plan you want to compare to the reserves plan you selected in the Summary window and click OK.
  7. In the Print Preview, select the folder or the entity you want to see a report for.
  8. Select the Report Depth.
  9. If you change any of the options above, click Preview to apply the change and view your selection.
