Configure AFENav and Integration

To import actual expenditures from AFENav or another source database into an entity’s capital cost forecast in , you must perform the following configuration:

  1. Create a view of your AFENav (or other source database) actual expenditures. Contact for this step.
  2. Add a “ValNavID” custom field to AFENav to hold the AFENav Vendor ID from and link AFEs to entities.
  3. Configure an External Data Connection to AFENav.
