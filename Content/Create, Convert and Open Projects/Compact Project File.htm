Compact Project File

Use this function with SQLite project files.

When you delete data in a SQLite project, the project file size is not automatically reduced because empty database pages still remain. The Compact Project File function removes empty database pages and defragments the project to reduce its size. For more information on this function, go to http://www.sqlite.org/lang_vacuum.html.

To compact a SQLIte project: In the Administration menu, select Data Cleanup > Compact Project File.
