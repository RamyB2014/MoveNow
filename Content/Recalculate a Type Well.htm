Recalculate a Type Well

You must recalculate the type curve after:

  • Editing the forecasts of the child wells (if you have used the forecast portion)
  • Excluding or including wells
  • Adding or removing wells
  • Editing the normalization of the child wells

To recalculate a Type Well curve

  1. From the Entity Type list in the main toolbar, select Type Wells.
  2. Select the type well.
  3. On the Type Well tab, click Recalculate.
