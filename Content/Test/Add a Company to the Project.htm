Add a Company to the Project

This procedure is a prerequisite to adding a company interest to an entity.

To add a company to a project

  1. On the Tools menu, point to Global Project Data and click Companies.
  2. In the Companies dialog box, click Add.
  3. Enter a Company Name.
  4. Enter a Default Working Interest.
  5. Select or deselect the following two options, as required:
    6. Option Description
    Enabled Enables or disables a company in the project. Only Enabled companies are included in economic calculations.
    Include in new entity interest If you enable this option, the company is automatically added to all entities (even previously created or imported wells) that have the default interest configuration, which means all wells with unedited interest configurations.
    If this option is enabled and you import wells into a project, the company will be added to the imported wells.
