Edit Facilities

In the Facility Editor, you can edit, add, or remove facilities.

To edit, add, or remove facilities

  1. On the Tools menu, point to Global Project Data, and select Facilities.
  2. Do one of the following:
    3. To Do this
    Edit a facility, Select a facility from the list and edit the Facility Properties.
    Add a facility, Click Add. Enter a name and click OK. Edit the Facility Properties.
    Remove a facility, Select a facility from the list and click Remove.
