User Options: Connection Settings

These settings determine the default connection settings for Oracle and SQL Server projects.

To configure the connection settings

  1. Complete the following fields for an Oracle and/or SQL Server connection:
    Field Description
    Preferred Oracle Connection
    Server Server name of the default Oracle server
    DBA User User name of the DBA for the Oracle connection
    Password Password for the DBA user
    Default Tablespace Default Oracle tablespace for project tables
    Default temporary tablespace Default Oracle tablespace for temporary tables
    Preferred SQL Server Connection
    Server Default SQL Server server name for project tables
  1. Click OK.
